## 3. 자재 4종 메커니즘 (검증 반영)

### 3.1 타일 ⭐⭐⭐⭐⭐

#### 입력
- 평수 ↔ 가로·세로(m) 토글
- 시공 부위 세그먼티드: **욕실/주방·바닥·일반벽** (로스 디폴트 자동 변경)
- 타일 규격: 300×300, 300×600, 600×600, 600×1200, 800×800, + 직접 입력(mm)
- 박스당 매수 (규격 선택 시 자동 채움 + 직접 수정 가능)
- 로스율 % — 프리셋 `7 / 10 / 15 / 20`

#### 산식
```
타일1매 ㎡ = (W_mm / 1000) × (H_mm / 1000)
순수 매수  = 시공㎡ / 타일1매
발주 매수  = ceil(순수 × (1 + 로스%))
박스 수    = ceil(발주 / 박스당매수)
```

#### 박스당 매수 자동 매핑 (★ 검증 반영)
| 규격 (mm) | 박스당 매수 |
| --- | --- |
| 300 × 300 | 16 |
| 300 × 600 | 8 |
| 600 × 600 | 4 |
| 600 × 1200 | 2 |
| 800 × 800 | 2 |
| custom | 직접 입력 |

#### 부위별 로스 디폴트 (★ 한국 플랫폼 절충)
- **일반벽 7%** · **바닥 10%** · **욕실·주방 15%**
- 사용자가 프리셋 버튼으로 즉시 변경 가능

#### 결과 출력
- 발주 권장 매수 (큰 숫자), 박스 수, 시공 면적, 순수 필요 매수

---

### 3.2 도배 ⭐⭐⭐⭐⭐ (간편/상세 토글)

#### 간편 모드 — `도배평 = 전용평 × 2.5` (한국 B2C 통설)
```
도배 평수 = 실평 × 2.5
도배 ㎡   = 도배평수 × 3.3058

롤당 시공평수:
  실크    5평/롤
  합지    2평/롤
  광폭합지 5평/롤

순수 롤 = 도배평수 / 롤당시공평수
발주 롤 = ceil(순수 × (1 + 로스%))
```
- 한국 마미견적서·꾸미고·세민직업학교 공통값 (×2.5)

#### 상세 모드 — `벽 + 천장 분리` (B 사이트 산식, 한국 통설 동등)
```
벽 도배 평수    = 실평 × 1.65 × (천장/2.4)
천장 도배 평수  = 실평 × 0.65   (천장 포함 시)
도배 평수      = 벽 + 천장
도배 ㎡        = 도배평수 × 3.3058

(이하 동일)
```
- B 사이트 검증식 "32평 천장포함 = 실크 15롤" 산식 적용 시 정확히 일치
- 천장 높이 보정 가능 (간편 모드엔 없음)

#### 공통 입력
- 시공 평수 (실평수)
- 벽지 종류: 실크 / 합지 / 광폭합지 / *(향후 점착메모리얼·뮤럴 추가 검토)*
- 천장 도배 포함 체크박스
- (상세 모드만) 천장 높이 m
- 추가 로스율 % — 프리셋 `0 / 10 / 15 / 20`

#### 풀 (kg) — 보수적 단일값으로 보정 (★ 검증 반영)
```
풀 = ceil(도배㎡ / 12)   ← 종류별 격차 없이 통일 (≈ 0.5~1kg/롤)
```
- 이전 B의 "실크/15, 합지/8"은 격차 과장이라 검증됨 → 단일 12로 통일
- 합지·실크 차이는 농도·희석비 차이지 풀 총량은 비슷

#### 1롤 표준 규격 (LX 공식 시드값)
| 종류 | 폭 | 길이 | 롤 면적 |
| --- | --- | --- | --- |
| 실크 | 1.06 m | 15.6 m | 16.5 ㎡ (≈ 5평) |
| 광폭합지 | 0.93~1.06 m | 12.5~17.75 m | 시리즈별 다름 |
| 소폭 합지 | 0.53 m | 12.5 m | 약 6.6 ㎡ (≈ 2평) |

#### 결과 출력
- 발주 권장 롤 수 (큰 숫자), 도배 평수, 도배 ㎡, 예상 풀 kg

### 3.3 마루 ⭐⭐⭐⭐⭐ (Q6 정답 = 1.0평/박스)

#### 입력
- 평수 ↔ 가로·세로 토글
- 마루 종류: 강화 / 강마루 / 원목 / 헤링본 / 직접 입력
- 박스당 평수 (종류 선택 시 자동 채움 + 직접 수정 가능)
- 실 둘레 m (걸레받이용)
- 로스율 % — 프리셋 `5 / 8 / 10 / 15`

#### 산식
```
순수 박스 = 평 / 박스당평수
발주 박스 = ceil(순수 × (1 + 로스%))
발주 평수 = 평 × (1 + 로스%)
```

#### 카탈로그 시드값 (★ 검증으로 결정)
| 종류 | 시드값 | 근거 |
| --- | --- | --- |
| 강화마루 | **0.5 평/박스** | 1.65㎡ = 0.5평 시장 표준 |
| **강마루** | **1.0 평/박스** | 구정·동화 표준 (장판나라 SKU 재확인) |
| 원목마루 | 0.5 평/박스 | 마루안 등 다수 SKU |
| 헤링본 | 0.5 평/박스 | 구정 프라하강 등 |

**중요**: 박스당 평수는 **net (박스 안 자재 실제 면적)**, 로스율은 별도 적용. 이중계산 없음.

#### 검증된 한국 표준 (영구 지식)
- LX 공식: `소요량 = 평수 × 1.1` → 로스 10%와 정확히 일치
- 강마루 1.0평/박스: 장판나라·동화·구정 모두 "1박스 1평" 표기
- 걸레받이 **1본 = 2440mm (8자)** MDF/PVC 모두

#### 차별화 슬롯 (Phase 2~)
- 걸레받이 본 환산: 둘레 m → 본 수
- 마루 본드 권장량

#### 결과 출력
- 발주 권장 박스 수 (큰 숫자), 시공 면적, 발주 평수, 걸레받이 m

---

### 3.4 페인트 ⭐⭐⭐⭐⭐ (간편/상세 토글)

#### 간편 모드 — `coverage 6 ㎡/L` (4대 제조사 공통)
```
벽 도장 ㎡  = 평 × 1.65 × (천장/2.4) × 3.3058
천장 도장   = 평 × 3.3058
직접 입력   = 직접 ㎡

도장면적 = 시공㎡ × 도장횟수
순수 L  = 도장면적 / 6   (수성 디폴트, LX·노루·삼화·오늘의집 공통)
발주 L  = ceil(순수 × (1 + 로스%))
말통    = ceil(발주 / 18)
```

#### 상세 모드 — 페인트 종류별 분리 + "이론 vs 실측" 두 줄 표기
| 종류 | 실측 coverage | (이론값 참고) |
| --- | --- | --- |
| 수성 | **6 ㎡/L** | 이론 10 (노루 공식 명시) |
| 친환경 | **6~7 ㎡/L** | 이론 8 |
| 유성 | **8 ㎡/L** | 거의 동일 |
| 퍼티/프라이머 | **5 ㎡/L** | OK |
- "이론도포량은 손실 미포함" 안내 (노루 공식 입장)

#### 공통 입력
- 시공 부위 세그먼티드: 벽 / 천장 / 직접 면적
- 실 평수 · 천장 높이 m
- 도장 횟수: 1 / 2 / 3 (디폴트 2 — 한국 표준)
- 로스율 % — 프리셋 `10 / 30` (붓·롤러 10% / 스프레이 30% — 노루 공식)
- (옵션 체크) 벽지 위 도장 → coverage 자동 -20~50% 보정 (오늘의집 발견값)

#### 1말통 = 18L (한국 표준, KCC·삼화·노루·제비스코 공통)

#### 결과 출력
- 발주 권장 말통 수 (큰 숫자), 시공 면적, 도장 면적, 순수 필요량 L, 4L 캔 환산

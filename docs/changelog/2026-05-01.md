# 2026-05-01 — 라운지(커뮤니티) v1 출시

수플렉스에 글로벌 메뉴 8번째 **"라운지"** 신설. 인테리어 회사의 디자이너·현장팀·운영진이 모이는 클로즈드 커뮤니티이며, **한 번이라도 승인된 회사에 속했던 사용자는 퇴사 후에도 평생 접근 가능**하도록 설계 — "인테리어 생태계 자산화" 비전의 첫 단계.

---

## 1. 기획·결정 (봉기님 컨펌)

`docs/커뮤니티_기획서.md` 단일 문서로 정리. 핵심 결정 11개:

- 메뉴 이름 **"라운지"**, 위치 상위 8번째 글로벌 메뉴
- 카테고리 6개: 공정·시공 노하우 / 수플렉스 사용 팁 / **스케치업 루비** / 자유잡담 / 구인구직 / 공지
- 직군·주제 `[]` 태그 화이트리스트(직군 3 + 주제 18)
- 회사명 노출 옵션 (기본 OFF, 닉네임 우선)
- **단가·시세 게시 허용** (클로즈드 커뮤니티의 힘)
- 거래처 실명 비방·클라이언트 식별 정보·AI 자문 요청 글 금지
- 루비 스크립트 면책: 사용자 책임, 신고 시 즉시 비공개 + 24h 처리 SLA
- 슈퍼어드민(봉기님)이 시드 게시 직접 작성, 베타 피드백도 라운지 안에서
- v3 비전: **프리셋 공유 마켓** — 최고등급 회사가 라벨·키워드·어드바이스 5묶음 공개·import

## 2. 데이터 모델 (8개 신설)

```
LoungeMembership   — userId 1:1, 회사와 무관하게 살아남음 (status: active|suspended|banned)
LoungePost         — 카테고리(6) · 본문(마크다운+코드 블록) · pinnedToHome 큐레이션 · soft delete
LoungeAttachment   — 이미지 + .rb 통합 (kind: image|ruby|other), Cloudinary 저장
LoungeComment      — 1단(대댓글 X), soft delete
LoungeReaction     — postId+userId unique, 토글
LoungeTag          — 화이트리스트(role|topic), 부팅 시 멱등 시드
LoungePostTag      — 글당 1~3개
LoungeReport       — 5사유, pending → resolved/dismissed
```

기존 도메인 모델과 격리(`userId` 외 FK 거의 없음) — 지출 모듈 격리 정책과 동일 사상.

## 3. 멤버십 자동 부여 hook

- 회사 승인(`/api/admin/companies/:id/approve`) → 그 회사 모든 멤버
- 초대 수락 3경로(좀비 복구·신규 가입·로그인 합류) → 회사가 APPROVED일 때만
- OWNER 직접 멤버 추가 → APPROVED일 때만
- 어드민 backfill 엔드포인트(멱등) — 베타 초기 보강용

`server/src/services/lounge.js`의 `grantIfCompanyApproved` 1개 함수로 통일.

## 4. 백엔드 API (`/api/lounge/*`)

베타 진입 통제 우회 분기 추가 + 자체 가드 `requireLoungeMember`. 슈퍼어드민 우회.

- `categories` `tags` `me` (GET/PATCH)
- `posts` 목록(category·tags·search·cursor) / CRUD / 본인 글만 수정·삭제
- `comments` 작성·수정·삭제 (1단)
- `reactions` 토글(POST 1번으로 ON/OFF)
- `reports` 글/댓글 신고 (5사유)
- `home-pinned` 1순위 큐레이션 → 2순위 7일 인기글 자동 fallback
- `attachments` 업로드(이미지 5MB×5장 / .rb 1MB×3개) · 다운로드(`/attachments/:id/download`로 Cloudinary URL 반환 + 카운트++) · 삭제

## 5. 프런트

- `/lounge` Lounge 페이지 — 카테고리 탭(전체+6) · 태그 필터(직군·주제 멀티) · 검색 · 글쓰기 모달 · 본인 라운지 프로필
- `/lounge/:postId` LoungePost — 본문(코드 블록 ```ruby 자동 렌더) · 첨부(이미지 갤러리 + .rb 다운로드 카드) · 댓글 1단 · 공감 토글 · 신고 모달
- `Layout` NAV 8번째 "라운지"
- `Dashboard` 상단에 `HomeLoungeTip` 카드 — 라운지 비멤버 자동 숨김, 24h dismiss

## 6. 어드민 콘솔 새 탭 "💬 라운지"

- 신고 큐 (처리 대기 / 전체)
- 액션: 숨김 / 기각 / 홈 큐레이션 / 멤버십 정지 / 복원
- 멤버십 일괄 보강 버튼 (APPROVED 회사 멤버 멱등 부여)
- 모든 액션 audit 로그 (`lounge.report_hide` 등)

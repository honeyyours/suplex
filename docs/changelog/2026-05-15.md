# 2026-05-15 — UI 다듬기·견적 그룹 합계·라운지 라벨 단축

오늘은 ★ 새 기능 3개(견적 그룹 합계, 견적 비고 키보드, 모바일 프로젝트 필터 시트)와 다수의 UI 다듬기. 인테리어 현장 운영자가 모바일에서 들고 다니며 쓰는 경험을 정렬하는 데 집중.

---

## 1. ★ 새 기능

### 1-A. 견적 — 그룹별 합계 표시
- `linesWithMeta`에 `_groupTotal`(그룹 내 라인 `quantity*unitPrice` 합) 부착
- **그룹 헤더 우측**: `합계 N원` 작은 배지 (총합 > 0 일 때만)
- **그룹 종료 마커**: 기존 `↩ 그룹 종료` → `↩ 그룹 합계 N원`으로 교체
- 그룹 아래 항목 추가/수정·수량·단가 변경 시 자동 재계산
- 종료 마커가 없는 그룹도 헤더에선 합계 노출

### 1-B. 견적 — 비고 셀 Enter 동작 분리
- **Enter** → 다음 행 같은 컬럼으로 이동 (마지막이면 새 행 추가) — 기존
- **Ctrl/⌘ + Enter** → 셀 안에서 줄바꿈 (네이티브 textarea 동작 허용)
- placeholder "(Ctrl+Enter 줄바꿈)"으로 안내 변경

### 1-C. 프로젝트 모바일 — 진행중 상단 영역 압축 (A안)
- 🧪 샘플 시연용 생성 버튼 모바일 hide (sm+에서만)
- 모바일 1행 = `[🔍 검색] + [⚙️ 필터]`
- ⚙️ 필터 탭 → 바텀시트로 연도·상태·정렬 한 번에 (초기화/적용 버튼)
- 디폴트(올해/전체/최근순)와 다르면 ● 점 표시
- 데스크탑은 기존 필터 카드 그대로

## 2. 캘린더 / 일정 정리

### 2-A. 월 구분 띠지 제거
- AggregateCalendar·ScheduleCalendar 두 곳에서 매월 1일 셀 `border-t-2 border-t-navy-400` 보더 제거 (가시성 저하)
- ScheduleCalendar에선 `col-span-7` "2026년 N월" 가로 헤더 띠지도 제거 + `weeks` 메모/Fragment 정리, 평탄한 `grid.map`으로 단순화
- M/1·navy-700 강조 텍스트는 정보성으로 유지

### 2-B. 모바일 캘린더 셀 밀집화
- Schedule 페이지 wrapper padding `p-1 sm:p-5` → `p-0 sm:p-5` (모바일 풀폭)
- AggregateCalendar / ScheduleCalendar mobile `min-h` 80/96 → 56
- 셀 내부 날짜 헤더·일정 컨테이너 padding·gap 0 수준
- 일정 칩 mobile font 10px → 9px (_01 사이즈)
- HomeWeekSchedule 셀은 반대로 90/112로 키워 다른 캘린더와 통일

### 2-C. 캘린더 상단 기간 표시 제거
- ScheduleCalendar의 "2026.05.20 ~ 2026.06.04 (총 16일)" 헤더 제거
- ProjectInfoCard / ProjectDetail에서 이미 기간 노출됨 — 중복
- 우측 키보드 팁(데스크탑만)·로딩 인디케이터는 유지

### 2-D. 일정 내 협력업체 태그 등록 UI 일시 하이드
- ScheduleCalendar → ScheduleEntry `showVendorButton` prop 제거 (data 노출은 유지)
- MobileScheduleSheet `SHOW_VENDOR_REGISTRATION = false` 상수 게이트
- 이미 태그된 vendor 표시(🏢 vendorName)는 유지
- 되돌리려면 prop 다시 추가 + 상수 true

### 2-E. 전체 일정 모바일 — 프로젝트 칩 시각 분리 (A안)
- `MobileProjectTab`: `border-b-2` 밑줄 → `rounded-full` 알약 칩
  - active: bg-navy-700 흰색
  - inactive: bg-gray-100 회색 보더
- 컨테이너: `border-b` 제거, `gap-1.5` 칩 간격
- Row 1(현장/견적/전체) 탭 vs Row 2(프로젝트) 칩이 다른 위계로 시각 분리

## 4. 자재별 산식 (※ [[interior-material-calc]]과 다른 부분만 강조)

### 4.1 타일 — A와 거의 동일
```
타일1매 ㎡ = (W/1000) × (H/1000)
순수 매수  = 시공㎡ / 타일1매
발주 매수  = ceil(순수 × (1+로스%))
박스 수    = ceil(발주 / 박스당 매수)
```
규격: 600×600 등, 로스 프리셋, custom 폭/높이 — A와 사실상 같음.

### 4.2 도배 — ★★★ A와 완전히 다른 접근
```
벽 도배 평수    = 실평 × 1.65 × (천장/2.4)       ← "32평 천장포함 = 실크 15롤" 검증
천장 도배 평수  = 실평 × 0.65                    ← 천장 포함 토글
도배 평수      = 벽 + (선택 시 천장)
도배 ㎡        = 도배평수 × 3.3058

롤당 시공평수:
  실크    = 5평/롤   (폭·길이가 아니라 시공평수 그대로 등록)
  합지    = 2평/롤
  광폭합지 = 5평/롤

순수 롤  = 도배평수 / 롤당시공평수
발주 롤  = ceil(순수 × (1+로스%))

풀 (kg):
  실크/광폭 = ceil(도배㎡ / 15)
  합지      = ceil(도배㎡ / 8)   ← 합지가 풀 더 먹는다는 실무 반영
```

**A와의 차이**
- A는 롤 면적(폭 m × 길이 m)으로 계산 → ㎡ 기반 → 학술적 정확
- B는 **"롤당 시공평수"** 표를 직접 — 한국 도배사 카탈로그·시공팀 입에서 그대로 나오는 단위
- B는 **천장 높이 보정** + **풀 종류별 분리** + **합지 풀 1.9배 더 먹는 사실** 반영 — **A보다 한 수 위**
- B는 "환산식에 자연 로스가 포함됨 → 추가 로스 0% 권장" 명시 → 사용자 인지 비용 ↓

### 4.3 마루 — 박스당 평수 표가 다름
| 종류 | A | B |
| --- | --- | --- |
| 강화마루 | (없음) | **0.7평/박스** |
| 강마루 (laminate) | 0.7평/박스 | 0.55평/박스 |
| 원목마루 | 0.5평/박스 | 0.45평/박스 |
| 헤링본 | 0.45평/박스 | (없음) |

- 같은 "강마루"가 A=0.7, B=0.55 → **카탈로그 진본 확인 필요**
- A는 헤링본 박스값을 별도 옵션으로, B는 로스율로만 처리

### 4.4 페인트 — A 산식에 천장 높이 보정 추가
```
벽 도장 ㎡  = 평 × 1.65 × (천장/2.4) × 3.3058   ← (천장/2.4) 보정이 추가됨
천장 도장   = 평 × 3.3058
직접        = 직접 입력 ㎡

도장면적 = 시공㎡ × 도장횟수
순수 L  = 도장면적 / coverage
발주 L  = ceil(순수 × (1+로스%))
말통    = ceil(발주 / 18)
```
- coverage: 수성 10, 유성 8, 친환경 8, 프라이머 5 — **A와 동일**
- 차이: 천장 높이 보정 추가 ((H/2.4)) → 2.7m 천장이면 +12.5% 면적

### 4.5 조명 — ★ 완전히 다른 모델 (W/평 직접 테이블)
A는 학술적: `㎡ × lux × 보정 / 90(LED lm/W)`
B는 실무적: `평 × W/평 직접 테이블 × 높이보정`

| 공간 | A (lux) | B (lux / W/평) |
| --- | --- | --- |
| 거실 | 220 | 200 / **6 W/평** |
| 침실 | 130 | 120 / **4 W/평** |
| 주방 | 400 | 400 / **10 W/평** |
| 욕실 | 160 | 150 / **5 W/평** |
| 공부방 | 450 | 400 / **10 W/평** |
| 사무실 | 750 | 700 / **14 W/평** |
| 카페 | 400 | 400 / **10 W/평** |
| 복도 | 120 | 120 / **4 W/평** |

- **B 방식이 더 빠르고 실무친화** — 시공사가 도면 보고 즉시 검증 가능
- **A 방식이 더 일반화 가능** — LED 효율 90→120 같은 가정 변경 시 자동 반영
- 둘 다 LED 7W / 10W 개수 환산은 동일

### 4.6 에어컨 — ★★★ B가 압도적으로 정교
```
baseKcal:
  가정용       250          (A=400)
  사무실      350          (A=450)
  카페·상가   450          (A=520)
  음식점       550          (A=620)

방향 보정 (dirMult):
  북 0.85 / 동 1.0 / 남 1.10 / 서 1.25   ← A는 0.95/1.0/1.08/1.15

천장 보정    = max(1, H/2.4)              ← A는 1 + max(0,H-2.4)×0.16
오픈형       × 1.15                       (A는 1.10)
탑층         × 1.15                       (A는 1.12)

requiredKcal = 평 × baseKcal × dirMult × 천장 × 오픈 × 탑층

표준 평형 = [6,7,8,10,13,15,17,18,20,22,25,27,30,36]   ← 한국 시판 평형 enum!

4-조합 분기:
  ┌─ 가정용 × 스탠드 → 거실 스탠드(전체 평형) + 방 벽걸이 7평형 × N대 (평수→방수 추정)
  ├─ 가정용 × 시스템 → 거실 카세트(5/7/10/13) + 방 매립 4평형 × N대
  ├─ 상업용 × 스탠드 → 평형별 단일/분리(stand+wall) 권장
  └─ 상업용 × 시스템 → 1HP ≈ 2,500 kcal/h, 표준 HP [1,1.5,2,2.5,3,4,5,6,8,10] 매칭
                       크기에 따라 단일/2대분산/3대분산 자동
```

**A 대비 우위**
- baseKcal이 A보다 **30~40% 낮음** — A는 너무 보수적(과대 산정), B가 실제 현장값에 가까움
- **시판 평형 enum**으로 매칭 → "권장 18평형" 같은 실제 시장 단위 출력
- **가정/상업 × 스탠드/시스템 4-조합 분기** → 각각 다른 시공 패턴 반영
- **HP 환산 (1HP=2500 kcal/h)** — 상업용 시스템 표준
- **평수→방수 추정** (12평↓=0실, 12~20=1실, …, 40↑=4실) — 한국 아파트 실정

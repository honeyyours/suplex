영역 설명

variant 배지 — variant별 색상(gray / navy / amber). 컴포넌트 상단에 작은 pill 형태. 라벨 텍스트: "준비 중" / "PRO 기능" / "베타 검증 중".
아이콘 — 4xl 이모지. comingSoon·planGate·betaCheck 다 큰 시각적 신호 역할.
제목·설명 — 제목은 18px 굵게. 설명은 14px 회색, max-w-md(약 420px)로 가운데 정렬.
예정/잠긴 기능 리스트 — comingSoon=예정 / planGate=잠긴. 흰 배경 + 회색 보더 카드 안에 ul로.
CTA 버튼 — planGate에서만 노출. navy-700 브랜드 컬러. "PRO로 업그레이드" 같은 행동 유도.
푸터 문구 — variant별 다른 한 줄 안내. "곧 구현 예정" / "베타 동안 무료, 정식 출시 시 활성" / "완성도를 끌어올리는 중".
compact 변형 — 한 줄 띠 형태. 페이지 헤더 위 또는 메뉴 옆에 부드럽게 부착. C 분류의 "노출하되 안내" 원칙 구현.

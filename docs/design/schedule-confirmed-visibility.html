SUPLEX · DESIGN BRIEF

일정 확정 표시 — 가시성 개선

Suplex의 캘린더(전체일정 · 프로젝트 일정 · 홈 주간 브리프)에서 일정 항목이 확정(confirmed) 상태일 때 체크 표시(✓)가 너무 작고 흐려서 한눈에 들어오지 않는다는 피드백입니다. 모바일 바텀시트(개별 일정 상세 화면)는 최근 알약 pill + 좌측 컬러바로 리뉴얼되어 충분히 눈에 띄지만, 달력 격자 안 작은 일정 막대에서는 여전히 ✓ 한 글자로만 표시됩니다. 이 작은 막대들에서도 확정/미확정이 한눈에 구분되도록 다듬어 주세요.

의뢰 핵심 — 디자이너가 자유롭게 풀어주세요. 아래는 시방서가 아닌 출발점입니다. ① 좁은 셀(모바일 한 칸 약 50px 너비) 안에서도 작동해야 함 ② 공정 색(배경/보더)과 충돌하지 않아야 함 ③ 협력업체 태그·삭제 아이콘과 겹치지 않아야 함 ④ 미확정도 함께 보일 때 위계가 분명해야 함.

① 영향 받는 화면 — 현재 상태

아래 네 화면은 모두 같은 ✓ 마커 패턴을 씁니다 — absolute right + text-emerald-600 + drop-shadow. 공정 색 배경 위에서 너무 작아 잘 안 보입니다.

② 이미 정돈된 짝 — 모바일 바텀시트

셀을 탭하면 올라오는 모바일 바텀시트는 최근 리뉴얼되었습니다 — 알약 형태 pill + "확정/미확정" 텍스트 라벨 + 좌측 4px 컬러바 조합으로 확정 상태가 매우 분명합니다. 이 의도(같은 시각 언어)를 셀 안 작은 막대에서도 살릴 수 있는 방안을 부탁드립니다 — 좁은 공간에서도 작동하는 압축된 변형이 필요합니다.

좋은 점 (유지): pill의 형태 변화(점선 → 실선·채워짐) + 텍스트 라벨(미확정 → 확정) + 좌측 컬러바가 한 번에 작동합니다. 이 의도를 캘린더 격자 셀의 좁은 막대에서도 살리려면 어떤 시각 도구를 골라야 할까요?

③ 풀어야 할 문제

  1. 좁은 모바일 셀(약 50px)에서도 "확정/미확정"이 한눈에 구분되어야 합니다. 텍스트 라벨은 들어갈 자리가 없습니다.
  2. 공정 색 배경과 충돌하지 않아야 합니다. 현재 막대 배경은 공정별 파스텔 톤(sky/pink/violet/amber/rose…), 전체일정 화면에서는 프로젝트별 색까지 더해져 시각 부담이 큽니다.
  3. 막대는 좌측 3px 보더 컬러바가 이미 공정/프로젝트 색을 점유하고 있습니다. 좌측 컬러바를 확정 시그널로 다시 쓰기는 어렵습니다 — 다른 위치/도구가 필요합니다.
  4. 웹 호버에서 우측에 삭제·협력업체 태그 아이콘 팝업이 뜹니다 — 호버 시 확정 시그널이 가려져도 됩니다(클릭 가능하면 됨).
  5. 미확정 일정도 함께 보일 때 위계가 분명해야 합니다 — 모든 일정이 똑같이 강조되면 의미가 없습니다.
  6. 확정/미확정 비율은 보통 3:7 정도(확정이 소수)입니다. 확정 쪽이 더 눈에 띄어야 합니다.

④ 가능한 방향 (예시 — 자유 변형)

  • 막대 배경 채도/투명도를 미확정은 낮추고 확정은 진하게 — 색 자체로 위계
  • 확정 시 막대 좌측 컬러바를 두껍게(3px → 5px) + 공정 색이 아닌 일관된 emerald로 전환
  • 확정 막대 전체 보더 외곽선 추가(공정 색 외곽선)
  • ✓ 아이콘을 좌측 prefix로 옮기고 배경에 작은 원형 chip으로 감싸기
  • 확정 막대만 약한 그림자/inset shadow로 "들려있는" 느낌
  • 미확정 막대를 점선 보더로 처리해 확정과 형태로 구분(모바일 시트 pill과 같은 언어)
위 항목은 자유 변형의 예시일 뿐입니다. 디자이너의 본 시각으로 풀어주세요 — 단일 도구든 조합이든 무방합니다.

  • 토글 방법 (웹): 막대 우클릭 → 컨텍스트 메뉴 "✓ 확정" / "✓ 확정 해제"

  • 막대 클릭 (웹): 인라인 텍스트 편집 모드 진입
  • 막대 클릭 (모바일): 무시 (셀 단위 탭만 동작)

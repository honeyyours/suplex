영역 설명

배지(BETA) — 카드 제목·메뉴 라벨 옆에 인라인 부착. amber 톤(`bg-amber-50 text-amber-700 border-amber-200`)으로 wedge 차원의 "검증 중" 신호. 클릭하면 팝오버 토글.
팝오버 헤더 — `[기능명] — 베타 검증 중`. 굵게, 14px 미만 작은 글꼴로 컴팩트하게.
안내 문구 — "완성도를 끌어올리는 중입니다. 사용해보시고 불편한 점·아이디어 보내주세요." 강요 X, 친근한 톤.
피드백 입력 — 3줄 textarea, 보내기 버튼은 navy-700(브랜드 컬러). 비어있으면 disabled 50%.
전송 완료 — 1.5초간 emerald 컨퍼메이션 후 자동 닫힘. 사용자 흐름 끊지 않음.

tone props로 amber / gray / navy 3종 지원. 디자이너가 다른 톤(예: violet, sky) 추가 제안 가능.

인터랙션

  • 배지 클릭: 팝오버 토글(open ↔ close). 부모 클릭 이벤트 전파 차단(`e.stopPropagation()`).
  • 팝오버 외부 클릭: 자동 닫힘(`mousedown` 이벤트 listener).
  • 피드백 입력 빈 상태: "보내기" 버튼 disabled (opacity 50%).
  • 전송 후: 1500ms 동안 emerald 메시지 → 자동 닫힘 + textarea 초기화.
  • 현재 시안: 콘솔 출력만. 정식 구현은 BetaFeedback API 신설 또는 라운지 어드민 모더 시스템과 통합.

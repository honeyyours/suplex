# 수플렉스(SUPLEX) 개인정보처리방침

**시행일**: 2026-04-30 (베타)
**버전**: 0.1 (베타 1차 초안)
**개인정보 보호책임자**: support@suplex.kr

> 본 처리방침은 클로즈 베타 단계의 1차 초안이며, 정식 출시 전 법률 자문을 거쳐 갱신될 예정입니다.

---

## 1. 수집하는 개인정보 항목

### 1-1. 회원가입 시 (필수)

- 이메일 주소
- 비밀번호 (단방향 암호화 저장 — bcrypt)
- 이름
- 회사명

### 1-2. 회원가입 시 (선택)

- 연락처
- 회사 사업자등록번호
- 회사 대표자명
- 회사 주소
- 회사 전화
- 회사 이메일

### 1-3. 서비스 이용 과정에서 자동 생성·수집

- 접속 로그 (접속 일시, IP 주소)
- 서비스 이용 기록 (감사 로그 — 중요 액션 16종 한정)
- 회원이 자발적으로 입력한 콘텐츠 (견적·메모·일정·사진 등 — 회원의 데이터로 분류)

## 2. 개인정보 수집 및 이용 목적

| 목적 | 항목 |
|---|---|
| 회원 식별 및 로그인 | 이메일, 비밀번호 |
| 서비스 제공 (견적서 갑지 자동 채우기 등) | 이름, 회사 정보 |
| 회원 응대 및 공지 | 이메일, 연락처 |
| 부정 이용 방지 및 보안 | IP 주소, 접속 로그, 감사 로그 |
| 서비스 개선 및 통계 분석 | 이용 기록 (개인 식별 정보 분리) |

## 3. 개인정보 보유 및 이용 기간

- **회원 탈퇴 시점까지** 또는 **법령상 보관 의무 기간** 중 더 긴 기간
- 회원 탈퇴 시 모든 개인정보는 즉시 파기되며, 다음 항목만 법령에 따라 일정 기간 보관됩니다:
  - 부정이용 기록: 1년 (정보통신망법)
  - 결제 관련 기록: 5년 (전자상거래법) — 베타 단계에서는 결제가 없으므로 해당 없음

## 4. 개인정보 제3자 제공

수플렉스는 회원의 동의 없이 개인정보를 제3자에게 제공하지 않습니다. 다만 다음의 경우는 예외로 합니다:

1. 회원이 사전에 동의한 경우
2. 법령에 의한 요청이 있는 경우

## 5. 개인정보 처리위탁 (해외 이전 포함)

서비스 운영을 위해 다음 외부 서비스에 일부 데이터 처리를 위탁합니다:

| 수탁업체 | 위탁 업무 | 처리 데이터 | 보관 위치 |
|---|---|---|---|
| **Neon (Postgres)** | 데이터베이스 호스팅 | 회원 정보·서비스 데이터 전체 | 미국 (AWS) |
| **Railway** | 서버 호스팅 | 서비스 처리 중 일시적 데이터 | 미국 |
| **Vercel** | 클라이언트 앱 배포·CDN | 정적 자산·HTTP 로그 | 미국 (글로벌 CDN) |
| **Cloudinary** | 사진 저장·전송 | 회원이 업로드한 사진 | 미국 |
| **Anthropic (Claude API)** | AI 비서 응답 생성 | AI비서에 입력된 질문 본문 (회원이 명시적으로 보낸 것에 한함) | 미국 |
| **Resend** *(정식 출시 후)* | 이메일 발송 | 수신자 이메일·메일 본문 | 미국 |

**해외 이전 안내**: 위 수탁업체 모두 데이터 보관 위치가 해외(미국)입니다. 회원이 서비스 가입 시 본 처리방침에 동의함으로써 해외 이전에 동의한 것으로 간주됩니다.

각 수탁업체와는 데이터 처리 위탁 계약을 체결하여 개인정보 보호 수준을 유지하도록 합니다.

## 6. 회원의 권리

회원은 언제든지 다음의 권리를 행사할 수 있습니다:

1. **열람**: 본인의 개인정보 조회 (서비스 내 설정 페이지)
2. **수정**: 부정확한 정보 수정 (서비스 내 설정 페이지)
3. **삭제**: 회원 탈퇴 시 모든 개인정보 즉시 파기
4. **처리 정지**: 특정 처리에 대한 정지 요청 (이메일 문의)

권리 행사는 서비스 내 기능 또는 support@suplex.kr 이메일을 통해 가능합니다.

## 7. 개인정보의 안전성 확보 조치

- **기술적 조치**: 비밀번호 단방향 암호화(bcrypt), HTTPS 전송, JWT 기반 인증, 세션 무효화 즉시 반영, IP 기반 감사 로그
- **관리적 조치**: 접근 권한 최소화 (대표자 외 직원 접근 통제), 외부 수탁업체와의 처리 위탁 계약
- **물리적 조치**: 서비스가 클라우드 기반이므로 별도 물리적 시설 없음. 클라우드 사업자의 보안 조치에 따름

## 8. 쿠키 및 유사 기술

- 본 서비스는 **로그인 유지 목적의 인증 토큰**을 브라우저 localStorage에 저장합니다.
- 추적·광고·분석 목적의 제3자 쿠키는 사용하지 않습니다.

# 챕터 2. 홈 화면 사용법

> 이 챕터를 읽고 나면 — Suplex 홈 화면을 이용해 회사의 오늘 할 일과 진행 중인 프로젝트 상태를 한눈에 파악할 수 있게 됩니다.

---

## 2.1 홈 화면이란

Suplex 로그인 직후 또는 좌측 메뉴 **홈** 클릭 시 도착하는 페이지입니다. 회사의 **활동 피드**와 **주간 공정 미리보기**, **진행 중인 프로젝트 카드**가 한 화면에 모입니다.

[스크린샷 자리: 홈 화면 — 전체 영역]

---

## 2.2 화면 구성

홈 화면은 크게 세 영역으로 나뉩니다:

### 영역 ① — 활동 피드 (좌측 또는 상단)

회사 전체의 최근 활동이 시간 역순으로 표시됩니다:

- 새 프로젝트 생성
- 견적 발행 (DRAFT → SENT)
- 마감재 변경 (특히 CHANGED 상태로 전환)
- 발주 등록·수령 완료
- 현장보고 작성
- 팀원 초대·역할 변경

> 💡 누가 무엇을 했는지 5초 안에 파악 가능합니다. 출근 직후 가장 먼저 볼 화면.

### 영역 ② — 주간 공정 미리보기 (중앙 또는 우측)

이번 주 진행 예정인 공정·작업이 일자별로 펼쳐집니다:

```
2026-05-15 (수)  강남 래미안 304호  철거 공사 시작
2026-05-15 (수)  도곡 자이 1102호   목공 진행
2026-05-16 (목)  강남 래미안 304호  철거 공사 마감
2026-05-17 (금)  강남 래미안 304호  전기 공사 시작
...
```

이 영역의 데이터는 각 프로젝트의 **공정 일정 탭(3번 탭)**에서 입력된 일정이 자동 집계된 결과입니다.

### 영역 ③ — 진행 중인 프로젝트 카드 (하단 또는 좌측)

진행 중(상태가 종료 아닌)인 프로젝트들이 카드 형태로 나열됩니다:

| 카드 정보 | 표시 |
|---|---|
| 프로젝트명 | `강남 래미안 304-1502` |
| 클라이언트 | `박상철` |
| 면적 | `30평` |
| 진행률 | (공정 완료 비율 또는 일정 진행률) |
| 다음 마일스톤 | `2026-06-26 입주 예정` |
| 견적 차수 | `1차 발송` 또는 `2차 진행 중` |
| ⚠️ 알림 | 발주 후 마감재 변경 같은 경고 |

카드 클릭 → 해당 프로젝트 상세 페이지로 이동.

---

## 2.3 자주 보게 되는 알림 ⚠️

홈 화면 또는 프로젝트 카드에 다음과 같은 경고 표시가 뜰 수 있습니다:

| 표시 | 의미 | 해야 할 일 |
|---|---|---|
| ⚠️ 발주 후 마감재 변경 | 거래처 발주 완료 후 그 마감재 사양이 바뀐 상태 | 거래처에 변경 통보 또는 재발주 |
| 🔴 데드라인 임박 | 공정 마감일이 3일 이내 | 일정 점검 |
| 🟡 미정 마감재 다수 | UNDECIDED 상태 자재가 많음 | 클라이언트와 결정 |
| 🔵 1차 견적 미발송 | DRAFT 상태로 일주일 이상 | 발송 또는 폐기 결정 |

각 알림의 자세한 처리는 해당 챕터에서 다룹니다 (발주 → 챕터 8, 일정 → 챕터 12).

---

## 2.4 홈에서 자주 하는 작업

### 출근 직후 5분 루틴

```
1. 활동 피드 — 어제 누가 무엇을 했는지 훑기 (1분)
2. 주간 공정 미리보기 — 오늘·내일 일정 확인 (1분)
3. ⚠️ 알림 카드 — 처리 필요한 경고 클릭해서 들어가기 (3분)
```

### 새 프로젝트 시작

홈 화면 어디에든 있는 **+ 새 프로젝트** 버튼 또는 좌측 메뉴 **프로젝트 → + 새 프로젝트**.

### 빠른 검색

상단 검색창 (있는 경우)으로 프로젝트명·고객명·모델명 검색 가능.

---

## 2.5 모바일 화면

홈 화면은 스마트폰에서도 동작하도록 반응형으로 설계되어 있습니다. 다만:

- ✅ 활동 피드 보기 / 알림 확인
- ✅ 프로젝트 카드 클릭으로 진입
- ⚠️ 인라인 엑셀 편집 (마감재) — 작은 화면에서 편집 가능하지만 권장하지 않음
- ⚠️ 견적 작성 — 데스크톱 권장

현장에서는 **현장보고 입력** (사진 첨부)과 **체크리스트 점검** 정도가 모바일 작업으로 적합합니다.

# 부록 C. 단축키 한눈에 보기

> 이 한 페이지를 인쇄하셔서 책상에 붙여두시면 매번 매뉴얼 찾아볼 일이 없습니다.

---

## 마감재 인라인 편집

| 키 | 동작 |
|---|---|
| **클릭** 또는 `F2` | 행 펼치기 (편집 모드) |
| `Tab` | 다음 필드 |
| `Shift` + `Tab` | 이전 필드 |
| `Enter` | 다음 필드 (textarea 제외 — textarea는 줄바꿈) |
| `↑` `↓` | 행 위·아래 이동 |
| `←` `→` | 필드 좌·우 이동 (커서가 입력 끝/처음일 때) |
| `Esc` | 펼친 행 닫기 |
| **마지막 필드에서 `Tab`** | **다음 행 자동 펼침** ← 가장 유용 |

**자동 저장**: 입력 후 1초 뒤. 행이 초록색 깜박임 = 저장 완료.

---

## 견적 인라인 편집

| 키 | 동작 |
|---|---|
| `Enter` | 같은 컬럼 다음 행 |
| `↑` `↓` | 같은 컬럼 위·아래 |
| `←` `→` | 셀 이동 |
| `Tab` | 다음 셀 |
| **마지막 행 비고에서 `Tab`** | **새 행 자동 추가** |

견적 라인 컬럼 순서: `품명 → 규격 → 수량 → 단위 → 단가 → 비고`.

## 일반

| 키 | 동작 |
|---|---|
| `Ctrl` + `S` | (브라우저 기본) — Suplex는 자동 저장이라 누르실 필요 없음 |
| `Ctrl` + `P` | (브라우저 기본 인쇄) — 견적·발주 PDF 출력 시 사용 |
| `Ctrl` + `F` | (브라우저 검색) — 마감재·견적 표 안 빠른 검색 |
| `Ctrl` + `Z` | (브라우저 기본) — 입력 중인 텍스트 되돌리기. 저장된 데이터는 복원 안 됨 |

## 자주 쓰는 작업의 빠른 경로

| 작업 | 빠른 경로 |
|---|---|
| 새 프로젝트 | 좌측 메뉴 → 프로젝트 → + 새 프로젝트 |
| 견적 PDF 출력 | 프로젝트 → 견적 탭 → 우측 상단 인쇄 버튼 → PDF로 저장 |
| 마감재 변경 → CHANGED | 행 클릭 → status 바꾸기 → Esc |
| 2차 견적 | 견적 탭 → 1차 SUPERSEDED → + 새 견적 또는 1차 복제 |
| 직전 차수와 비교 | 2차 견적 화면 → "직전 차수와 비교" 버튼 |
| 영수증 입력 | 지출관리 → + 새 지출 (또는 프로젝트 → 지출 탭) |
| 팀원 추가 | 설정 → 팀관리 → + 팀원 추가 |
| 데이터 백업 | 설정 → 데이터 백업 → 다운로드 |

## 다크모드 토글

좌측 사이드바 하단 또는 설정에서 `light` / `dark` / `system` 선택.

| 모드 | 추천 시점 |
|---|---|
| `light` | 밝은 사무실, 점심 시간 |
| `dark` | 야근, 어두운 환경, 모니터 밝기 높을 때 |
| `system` | 잘 모르겠으면 이걸로 두세요 |

# 챕터 11. 메모

> 이 챕터를 읽고 나면 — Google Keep 스타일의 카드 메모로 프로젝트 관련 잡다한 정보를 빠르게 저장하고 분류할 수 있게 됩니다.

---

## 11.1 메모란

프로젝트의 잡다한 메모를 카드 형태로 저장하는 공간. **Google Keep과 비슷한 UX**입니다.

### 왜 별도 메모 탭이 있나

견적·마감재·발주 같은 정형 데이터로 표현하기 어려운 정보가 많습니다:

- 첫 미팅 노트 (클라이언트 취향, 라이프스타일)
- 협상 과정에서 오간 대화 캡처
- 거래처 추천 자재 (아직 결정 안 함)
- 클라이언트의 즉흥 요청 ("벽에 그림 걸 자리 두 군데")

이런 것들을 **마감재·견적에 억지로 넣으면 데이터가 더러워집니다**. 메모 탭에 카드로 저장.

---

## 11.2 카드 작성

프로젝트 → **메모** 탭 (10번 탭) → **+ 새 메모**.

### 카드 구성

| 필드 | 예시 |
|---|---|
| 제목 | "첫 미팅 노트 (5/10)" |
| 내용 | 자유 텍스트, 마크다운 일부 지원 |
| 태그 | "미팅", "디자인", "클라이언트 요청" 등 |
| 색상 | 카드 배경색 (분류용) |
| 핀 (고정) | 상단 고정 |

[스크린샷 자리: 메모 탭 — Google Keep 스타일 카드 그리드]

---

## 11.3 활용 예시

### 첫 미팅 노트 카드

```
제목: 첫 미팅 노트 (5/10 토)
태그: 미팅, 클라이언트 요청

[클라이언트 정보]
- 4인 가족 (부부 + 초등 자녀 2명)
- 입주 6주 후 (6/26 목표)
- 예산 5,000만 원 (상한 5,500)

[취향]
- 따뜻한 톤 선호 (와이프)
- 화이트 베이스 + 메이플 강마루
- 욕실은 그레이 / 화이트 무광

[가족 라이프스타일]
- 자녀 학용품 보관 공간 필요 → 작은방-1 책상 + 책장
- 부부 침실 화장대 → 기성품 직접 구매 예정
- 주방 식기세척기 빌트인 (와이프 강력 희망)

[다음 액션]
- 견적 1차 5/12까지 발송
- 강마루 샘플 5개 가져가기 (5/15)
```

### 거래처 추천 카드

```
제목: 욕실 타일 후보 (검토 중)
태그: 자재 검토

1. 동화자연마루 — 600×300 화이트 무광 (35,000/m²)
2. KCC 글라스 — 400×400 그레이 (32,000/m²) ← 와이프 마음 끌림
3. 윤현상재 — 300×600 베이지 (45,000/m²) ← 너무 비쌈

5/14까지 결정 받기로 함
```

### 즉흥 요청 카드

```
제목: 5/13 통화 메모
태그: 클라이언트 요청

박상철님 통화 내용 (오후 3시):
- 거실 벽에 액자 걸 자리 2곳: 소파 위, TV 옆
- 콘센트 추가: 거실 바닥 (2구), 안방 침대 좌우 각 1구
- 인터넷 단자: 작은방-1 책상 자리 추가 필요

→ 다음 견적 차수에 반영
```

## 11.4 정리 팁

- **태그 일관성**: 회사 내 태그 표준 정해두기 ("미팅", "클라이언트 요청", "거래처", "이슈" 등)
- **고정 카드**: 항상 봐야 할 카드 (예: 클라이언트 기본 정보)는 핀으로 상단 고정
- **색상 활용**: 빨강 = 긴급, 노랑 = 검토 중, 초록 = 완료 등 회사 룰 정해두기

# 챕터 16. 설정

> 이 챕터를 읽고 나면 — 회사 정보, 견적 기본비율, 마감재·견적 라인 템플릿, 공종 키워드 룰, 공정 데드라인 룰, 가전 규격 마스터, 데이터 백업 등 회사 단위의 모든 설정을 다룰 수 있게 됩니다.

설정은 **한 번 잘 해두면 평생 편한 작업**이 모여있는 곳입니다.

---

## 16.1 설정 메뉴 구조

좌측 메뉴 → **설정**. 다음 섹션이 있습니다:

| 섹션 | 무엇 | 사용 빈도 |
|---|---|---|
| 회사 정보 | 회사명·로고·도장 등 | 도입 직후 1회 |
| 견적 기본비율 | 디자인비·부가세 등 12개 % | 분기 1회 점검 |
| **마감재 템플릿** | 회사 표준 마감재 SET | **초기 + 자주 갱신** |
| **견적 라인 템플릿** | 회사 표준 공정 라인 | **초기 + 자주 갱신** |
| 공종 키워드 룰 | "도배" → workType=WALLPAPER 자동 매핑 | 분기 1회 |
| 공정 데드라인 룰 | "철거 = 2일" 같은 표준 기간 | 분기 1회 |
| **가전 규격 마스터** | 모델명 → 사이즈 자동 채움 데이터 | **자주 갱신** |
| 내 계정 | 본인 비밀번호·연락처 | 필요 시 |
| 데이터 백업 | 전체 데이터 다운로드 | 월 1회 |

---

## 16.2 견적 기본비율 (12개 %값)

회사가 평소 사용하는 비율을 미리 등록하면, 새 견적 만들 때 자동으로 채워집니다.

| 항목 | 일반 권장 |
|---|---|
| 디자인 및 감리비 | 10% |
| 부가가치세 | 0% (별도 안내) 또는 10% |
| 간접재료비 | 자재비 × 5% |
| 간접노무비 | 노무비 × 12% |
| 산재보험료 | 노무비 × 3.7% |
| 고용보험료 | 노무비 × 1.3% |
| 국민건강보험료 | 노무비 × 3.5% |
| 국민연금보험료 | 노무비 × 4.5% |
| 노인장기요양보험료 | 건강보험료 × 12% |
| 안전관리비 | 직접공사비 × 1.7% |
| 일반관리비 | 합계 × 6% |
| 이윤 | 합계 × 10% |

값은 시장가·회사 영업 정책에 따라 조정.

## 16.3 가전 규격 마스터

가전 모델명 → 사이즈·걸레받이 자동 채움 데이터.

이 데이터는 **단계적으로 시드되고 있습니다** ([appliance-db-plan.md](../../appliance-db-plan.md) 참고). 사용자가 직접 추가할 수도 있고, Suplex 시스템 시드분을 그대로 쓸 수도 있습니다.

### 모델 추가

| 필드 | 입력 |
|---|---|
| 모델 코드 | `LG DUE6BGL2E` (unique) |
| 브랜드 | LG |
| 카테고리 | 식기세척기 |
| 가로 (mm) | 598 |
| 높이 (mm) | 815 |
| 깊이 (mm) | 567 |
| 걸레받이 (mm) | 150 |
| 통풍 간극 (mm) | (있으면) |
| 출처 | LG 공식 사이트 |
| 검증 상태 | VERIFIED / PENDING / DISPUTED / USER_CORRECTED |

### 활용

마감재(APPLIANCE) 행에 모델 코드를 입력하면 사이즈가 자동 채워집니다. 매칭 안 되면 직접 입력.

## 16.4 데이터 백업

**설정 → 데이터 백업** → **전체 다운로드**.

JSON 파일로 회사 전체 데이터가 다운로드됩니다 (프로젝트·마감재·견적·발주·지출 모두 포함).

| 권장 빈도 |
|---|
| 월 1회 정기 백업 |
| 분쟁 직전·직후 즉시 백업 |
| 회사 변경(서버 이전 등) 시 백업 |

> 💡 백업 파일은 **회사 외부 저장소**(개인 클라우드, USB 등)에도 보관 권장. Suplex 서버 장애 시에도 데이터 잃을 일 없도록.

---

## 16.5 내 계정

본인 비밀번호 변경, 연락처 수정.

비밀번호 변경 절차:
1. 현재 비밀번호 입력
2. 새 비밀번호 입력 (8자 이상)
3. 새 비밀번호 확인 입력
4. 변경 클릭

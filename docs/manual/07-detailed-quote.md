# 챕터 6. 상세 견적서

> 이 챕터를 읽고 나면 — 한국 인테리어 표준 양식(갑지 + 18행 원가내역서)에 따라 상세 견적을 작성하고, 자재비·노무비·경비를 공종별로 입력하며, 간편 견적과 어느 걸 쓸지 판단할 수 있게 됩니다.

---

## 6.1 상세 견적이란

상세 견적은 **한국 건설·인테리어 업계의 표준 양식**입니다. 입찰·관급공사·대규모 공사에서 요구되는 형식.

### 구조

```
┌─────────────────────────────────────────┐
│  [1] 갑지 (표지)                         │
│   - 공사명, 발주처, 도급자, 공사기간      │
│   - 총공사비 (큰 글씨)                   │
│   - 도급자 인장                          │
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  [2] 원가내역서 (18행 표준)              │
│   1. 직접재료비                          │
│   2. 간접재료비                          │
│   3. 직접노무비                          │
│   4. 간접노무비                          │
│   5. 산재보험료                          │
│   6. 고용보험료                          │
│   7. 국민건강보험료                      │
│   8. 국민연금보험료                      │
│   9. 노인장기요양보험료                  │
│   10. 안전관리비                         │
│   11. 환경보전비                         │
│   12. 기타경비                           │
│   13. 일반관리비                         │
│   14. 이윤                               │
│   15. 부가가치세                         │
│   ...                                   │
│   ────────                              │
│   합계                                   │
└─────────────────────────────────────────┘
┌─────────────────────────────────────────┐
│  [3] 공종별 내역서                        │
│   공종 / 자재비 / 노무비 / 경비 / 합계    │
│   - 가설공사                              │
│   - 철거공사                              │
│   - 목공공사                              │
│   - 조적·미장공사                         │
│   - 도배공사                              │
│   - 바닥공사                              │
│   - 타일공사                              │
│   - 도장공사                              │
│   - 전기공사                              │
│   - 설비공사 (수도)                       │
│   - 가구공사                              │
│   - 가전공사                              │
│   - 청소·기타                             │
└─────────────────────────────────────────┘
```

## 6.2 어떻게 작성하나

프로젝트 → **상세 견적** 탭 (2번 탭) → **+ 새 견적**.

### 입력 화면

상세 견적은 간편 견적과 다르게 **공종별 그룹**으로 입력합니다:

[스크린샷 자리: 상세 견적 작성 화면 — 공종별 그룹]

### 공종 (workType)

`enum WorkType` 으로 정의된 표준 공종:

| 공종 (영문) | 한글 |
|---|---|
| TEMPORARY | 가설 |
| DEMOLITION | 철거 |
| WOODWORK | 목공 |
| MASONRY | 조적·미장 |
| WALLPAPER | 도배 |
| FLOORING | 바닥 |
| TILE | 타일 |
| PAINTING | 도장 |
| ELECTRIC | 전기 |
| PLUMBING | 설비·수도 |
| FURNITURE | 가구 |
| APPLIANCE | 가전 |
| CLEANING | 청소·기타 |

각 공종 안에 여러 라인을 추가:

| 항목명 | 규격 | 단위 | 수량 | 자재비 | 노무비 | 경비 | 합계 |
|---|---|---|---|---|---|---|---|
| 거실 강마루 | 메이플색 8.2T | m² | 18 | 75,000 | 20,000 | - | 1,710,000 |
| 안방 강마루 | 메이플색 8.2T | m² | 12 | 75,000 | 20,000 | - | 1,140,000 |
| ... | ... | ... | ... | ... | ... | ... | ... |

### 자재비 / 노무비 / 경비 분리

상세 견적의 핵심은 **금액을 세 가지로 쪼개서 입력**한다는 점입니다:

| 분류 | 의미 | 예시 |
|---|---|---|
| **자재비** | 자재 자체 가격 | 강마루 75,000원/m² |
| **노무비** | 인건비 (시공 인력) | 시공비 20,000원/m² |
| **경비** | 자재·노무 외 비용 | 운반비, 폐기물처리비 등 |

이렇게 분리하면:
- 부가세·간접비 계산 시 자재비 기준 / 노무비 기준 다르게 적용 가능
- 견적서가 매우 정밀해 신뢰도 ↑
- 입찰서 양식 대응 가능

> 💡 회사가 평소에 자재비·노무비·경비를 분리해서 관리하지 않으셨다면, 일단 자재비에 모두 넣고 노무비는 0으로 두셔도 됩니다 — 견적서는 정상 출력됩니다. 다만 점진적으로 분리해 입력하시는 게 좋습니다.

---

## 6.3 갑지 정보

상세 견적은 **갑지(표지)**가 필수입니다. 다음 정보를 입력:

| 항목 | 입력 |
|---|---|
| 공사명 | 보통 프로젝트명 그대로 |
| 발주처 | 클라이언트 (회사 또는 개인) |
| 도급자 | 본인 회사 |
| 공사 위치 | 현장 주소 |
| 공사 기간 | 시작~종료 |
| 총공사비 | 자동 계산 |

이 정보 대부분은 프로젝트 정보에서 자동으로 채워집니다.

---

## 6.4 자동 계산 — 18행 원가내역서

각 비율값은 **설정 → 견적 기본비율**에서 회사 기본값으로 미리 등록 가능. 견적별로 재정의할 수도 있습니다.

기본 비율 권장값 (한국 인테리어 시장 평균):

| 항목 | 일반 권장 % |
|---|---|
| 간접재료비 | 자재비 × 5% |
| 간접노무비 | 노무비 × 10~15% |
| 산재보험료 | 노무비 × 3~4% |
| 고용보험료 | 노무비 × 1.3% |
| 국민건강보험료 | 노무비 × 3~4% |
| 국민연금보험료 | 노무비 × 4.5% |
| 노인장기요양보험료 | 건강보험료 × 12% |
| 안전관리비 | 자재+노무비 × 1.7~2% |
| 환경보전비 | 직접공사비 × 0.3% |
| 기타경비 | 직접공사비 × 5~7% |
| 일반관리비 | 위 모든 합 × 6% |
| 이윤 | 위 모든 합 × 8~15% |
| 부가가치세 | 위 모든 합 × 10% |

> ⚠️ 이 비율은 **회사·공사 종류·발주처에 따라 크게 달라집니다.** 위 값은 참고용이며, 본인 회사의 영업 정책·시장가격에 맞게 조정해주세요.

## 6.5 마감재와의 연결

상세 견적도 마감재 탭의 항목들과 연결되지만, 간편 견적보다는 **느슨한 연결**입니다:

- 상세 견적은 공종별 그룹이라 spaceGroup과 자동 매핑되지 않음
- 자재비 라인을 입력할 때 마감재 탭에서 등록된 항목을 참고하시되, 직접 입력 필요

> 💡 시간이 지나면서 Suplex가 마감재 → 상세견적 자동 매핑 기능을 추가할 수도 있습니다 (현재 미구현). 일단 수동 작성 권장.

---

## 6.6 PDF 출력

브라우저 인쇄 스타일로 출력. 절차는 간편 견적과 동일:

1. 우측 상단 인쇄 버튼
2. 미리보기 확인
3. PDF로 저장
4. 외부 채널로 클라이언트에게 발송

## 6.7 간편 vs 상세 — 어느 걸 쓰나

이 질문이 매뉴얼에서 가장 자주 받는 질문입니다.

### 결정 기준

| 상황 | 추천 |
|---|---|
| 공사 규모 < 1,000만원 | 간편 |
| 공사 규모 1,000~5,000만원 | 간편 (보통) |
| 공사 규모 > 5,000만원 | 상세 (필요 시 둘 다) |
| 일반 가정 클라이언트 | 간편 |
| 사업자 클라이언트 (사옥·매장) | 상세 |
| 입찰·관급공사 | **상세 필수** |
| 첫 미팅 / 상담 단계 | 간편 (빠른 산출) |
| 합의 후 정식 견적 | 상세 (필요 시) |
| 변경 빈번한 프로젝트 | 간편 (변경 트래킹 더 쉬움) |

### 두 견적 같이 쓰는 패턴

큰 프로젝트는 **두 가지를 모두** 운용하기도 합니다:

```
[첫 미팅]
   ↓
[간편 견적 v1] — 빠르게 받아본 후 협상
   ↓ (합의)
[상세 견적 v1] — 정식 계약용
   ↓ (변경)
[간편 견적 v2] — 변경 후 빠른 합의
   ↓ (확정)
[상세 견적 v2] — 변경 정식판
```

# 챕터 1. 가입과 회사 설정

> 이 챕터를 읽고 나면 — 회사 계정을 만들고, 회사 기본 정보·견적 기본비율·마감재 템플릿을 등록한 상태가 됩니다. 이 작업은 회사당 **한 번만** 하시면 됩니다.

---

## 1.1 계정 만들기

### 가입 절차

Suplex 메인에서 **회원가입** 버튼을 누르면 다음 화면이 나옵니다:

[스크린샷 자리: 회원가입 화면]

| 입력 항목 | 설명 | 필수 |
|---|---|---|
| 회사명 | 사업자등록증 기준의 회사명 | ✅ |
| 대표자 이름 | 회사 대표 본인 이름 | ✅ |
| 이메일 | 로그인 ID로 사용. **이메일 1개당 회사 1개** | ✅ |
| 비밀번호 | 8자 이상 | ✅ |
| 연락처 | 010-XXXX-XXXX | 선택 |

**가입 버튼**을 누르면 자동으로:
- 회사(Company) 생성
- 가입자가 그 회사의 **OWNER**로 등록 (자동)
- 회사 마감재 기본 템플릿이 시드됨 (Suplex 기본값. 추후 회사 맞춤으로 다듬음)
- 홈 화면으로 진입

> ⚠️ 한 이메일 주소로 한 회사만 만들 수 있습니다. 회사가 두 개 있으시면 이메일을 둘 사용해주세요 (예: `대표자@회사A.com`, `대표자+회사b@회사A.com` 같은 alias 사용 가능).

### 로그인

이미 가입하셨으면 **로그인** 버튼:

| 입력 항목 | 설명 |
|---|---|
| 이메일 | 가입 시 입력한 주소 |
| 비밀번호 | 8자 이상 |

> 💡 비밀번호를 잊으셨다면 회사의 OWNER가 **팀관리 → 비밀번호 초기화**로 새 임시 비번을 발급할 수 있습니다. 본인이 OWNER이고 본인 비밀번호를 잊으셨다면 Suplex 운영팀에 문의 부탁드립니다 (이 경우는 외부 처리 필요).

## 1.2 회사 정보 등록

가입 직후 가장 먼저 하실 일은 **회사 정보 정확히 등록**입니다. 이 정보가 견적서 PDF 머리에 들어갑니다.

**좌측 메뉴 → 설정 → 회사 정보**

[스크린샷 자리: 설정 → 회사 정보 화면]

### 입력 항목

| 항목 | 입력 예시 | 어디에 쓰이나 |
|---|---|---|
| 회사명 (정식) | `(주)ABC인테리어` | 견적서·발주서 헤더 |
| 회사명 (간단) | `ABC인테리어` | 매뉴얼·메뉴 |
| 사업자등록번호 | `123-45-67890` | 견적서·세금계산서 |
| 대표자명 | `홍길동` | 견적서 |
| 회사 주소 | 사무소 주소 | 견적서 푸터 |
| 회사 연락처 | `02-1234-5678` | 견적서 푸터 |
| 이메일 | 대표 이메일 | 견적서 푸터 |
| 회사 로고 (이미지) | PNG/JPG 업로드 | 견적서 헤더 좌측 |
| 도장 이미지 (선택) | PNG (배경 투명) | 견적서 우측 하단 |

> 💡 **회사 로고와 도장은 견적서 외관의 95%를 결정합니다.** 정성껏 등록해주세요. 로고는 가로 약 200px, 도장은 정사각형 약 80×80px가 보기 좋습니다.

## 1.3 ★ 회사 템플릿 — 마감재 기본 SET (가장 중요한 초기 작업)

### 왜 중요한가

새 프로젝트를 만들 때마다 마감재를 처음부터 입력하면 시간이 너무 오래 걸립니다. **회사가 평소에 자주 쓰는 마감재 SET**을 한 번 등록해두면, 이후 모든 새 프로젝트가 이 SET을 자동으로 시작점으로 갖게 됩니다.

### 어디서 관리하나

**설정 → 마감재 템플릿** (또는 비슷한 이름)

이 화면에서 회사 표준 마감재를 행 단위로 추가·수정·삭제·활성/비활성 처리할 수 있습니다.

### 등록 권장 항목 — 30평 아파트 표준 SET 예시

이 표를 참고로 사용자 회사가 평소에 쓰는 마감재를 등록해주세요. 빈 칸은 신규 프로젝트마다 채워야 하지만, 항목 자체가 미리 등록되어 있으면 입력 시간이 70% 줄어듭니다.

> 💡 **`active` 플래그**: 한때 자주 썼지만 지금은 잘 안 쓰는 항목(예: 단종된 자재)은 active = false로 두세요. 삭제하면 과거 프로젝트의 참조가 깨질 수 있습니다.

### 견적 라인 템플릿도 함께

마감재 템플릿 외에 **견적 라인 템플릿**도 같은 메뉴에서 관리합니다. 회사가 자주 쓰는 공정 단위 견적 라인:

| workType | itemName | spec | unit | 자재비 | 노무비 | 경비 |
|---|---|---|---|---|---|---|
| DEMOLITION | 철거 공사 | 30평 일괄 | 식 | - | 2,500,000 | - |
| WOODWORK | 목공 + 가벽 | 거실+방 | 식 | 1,800,000 | 2,500,000 | 500,000 |
| WALLPAPER | 도배 공사 | 실크 벽지 | m² | 12,000 | 13,000 | - |
| FLOORING | 바닥 공사 | 강마루 | m² | 75,000 | 20,000 | - |
| ELECTRIC | 전기 공사 | 30평 일괄 | 식 | 600,000 | 600,000 | - |
| ... | ... | ... | ... | ... | ... | ... |

이렇게 등록해두면 새 견적 작성 시 **항목 선택만 하면 자재비·노무비·경비 자동 채움**되어 견적 시간이 절반 이상 줄어듭니다.

### 회사 견적 기본비율

**설정 → 견적 기본비율** (또는 비슷한 이름)에서 다음 12개 비율값을 회사 기본으로 등록해두실 수 있습니다:

- 디자인 및 감리비 (%)
- 부가세 (%)
- 간접재료비 (%)
- 간접노무비 (%)
- 산재보험료 (%)
- 안전관리비 (%)
- 일반관리비 (%)
- 이윤 (%)
- 등등

새 견적 만들 때 이 값들이 자동으로 채워집니다 (사용자가 견적별로 변경 가능).

[스크린샷 자리: 설정 → 견적 기본비율 화면]

## 1.4 팀원 추가하기

OWNER가 가입하면 회사의 모든 권한을 갖습니다. 팀원이 있다면 다음 절차로 추가:

**설정 → 팀관리 → 팀원 추가**

| 입력 항목 | 설명 |
|---|---|
| 이름 | 팀원 이름 |
| 이메일 | 팀원 본인 이메일 |
| 역할 | OWNER / DESIGNER / FIELD 중 선택 |

**역할별 권한**:

| 역할 | 권한 |
|---|---|
| **OWNER** | 모든 기능. 팀원 추가/삭제, 회사 정보 수정 |
| **DESIGNER** | 프로젝트·견적·마감재 작성·수정 |
| **FIELD** | 현장보고 입력, 체크리스트 점검, 마감재 조회 |

**추가 후**: 임시 비밀번호가 발급됩니다. 팀원에게 이메일/카카오톡으로 전달:

> "ABC인테리어 Suplex 계정이 만들어졌습니다.
> ID: 본인 이메일
> 임시 비밀번호: AB12cd34
> 첫 로그인 후 비밀번호를 변경해주세요."

자세한 흐름은 [Playbook #6: 신입 사원 들어왔을 때](playbooks/06-team-onboarding.md) 참고.

## 1.5 협력업체 (거래처) 등록

자주 거래하는 거래처(자재·인력·시공)를 미리 등록해두면 발주·지출 입력이 빨라집니다.

**설정 → 협력업체** (또는 팀관리 안)

| 항목 | 입력 예시 |
|---|---|
| 거래처명 | `XYZ타일` |
| 종류 | 자재 / 인력 / 시공 / 종합 |
| 대표자 | `김대표` |
| 연락처 | `010-XXXX-XXXX` |
| 사업자번호 | `123-45-67890` |
| 주거래 항목 | `타일·세라믹` |
| 비고 | 결제 조건, 특이사항 |

이 정보가 등록되어 있으면, 발주 입력 시 거래처를 검색해서 자동으로 정보가 채워집니다.

## 1.6 초기 설정 체크리스트

| ☐ | 항목 | 권장 시점 |
|---|---|---|
| ☐ | 회사 정보 (이름·사업자번호·주소·연락처) | 가입 직후 |
| ☐ | 회사 로고 업로드 | 가입 직후 |
| ☐ | 도장 이미지 업로드 | 가입 직후 |
| ☐ | 마감재 템플릿 30~50항목 등록 | 첫 주 |
| ☐ | 견적 라인 템플릿 20~30항목 등록 | 첫 주 |
| ☐ | 견적 기본비율 12개 값 등록 | 첫 주 |
| ☐ | 팀원 추가 (있다면) | 첫 주 |
| ☐ | 협력업체 5~10개 등록 | 첫 달 |

이 체크리스트를 다 끝내시면 **첫 프로젝트 견적 작성이 5분 안에** 끝납니다.

# Quick Start — 10분 안에 첫 견적 보내기

> 이 문서를 읽고 나면 — Suplex에 가입하고, 가상의 프로젝트를 하나 만들고, 첫 간편 견적서 PDF를 출력해서 클라이언트에게 보내는 일까지 끝납니다.

## 시작하기 전에 (30초)

이 문서가 필요한 분:
- 회사가 막 Suplex를 도입했고, 오늘이 첫날인 사용자
- 어디부터 손대야 할지 막막한 견적 담당자
- "데모 한 번 보고 싶은데" 하시는 사장님

준비물:
- 컴퓨터 한 대 (스마트폰으로도 가능하지만 화면이 좁아 권장하지 않습니다)
- 회사 정보 (회사명, 대표자명, 이메일)
- 가상의 프로젝트 정보 한 세트 (밑에서 예시 드립니다)

소요 시간: **약 10분**

---

## 1단계 — 가입 (1분)

1. 브라우저에서 Suplex 사이트로 이동합니다.
2. 우측 상단 **"회원가입"** 버튼을 누릅니다.
3. 다음을 입력합니다:

| 항목 | 입력 예시 | 비고 |
|---|---|---|
| 회사명 | `ABC인테리어` | 사업자등록증의 회사명 |
| 대표자 이름 | `홍길동` | 대표자 본인이 가입할 때 |
| 이메일 | `hong@abcinterior.com` | 로그인 ID로 사용됨 |
| 비밀번호 | (8자 이상) | |
| 연락처 | `010-1234-5678` | 선택 |

4. **"가입하기"** 클릭 → 자동으로 홈 화면으로 진입합니다.

> 💡 **자동으로 일어나는 일들**
> - 회사가 새로 생성됩니다 (Company)
> - 가입자가 그 회사의 **대표(OWNER)**로 등록됩니다
> - 회사 마감재 템플릿이 자동으로 시드됩니다 (기본 SET)
>
> 즉, 가입만 하면 빈 화면이 아니라 바로 일을 시작할 수 있는 상태가 됩니다.

[스크린샷 자리: 가입 화면 — 입력 폼]

---

## 2단계 — 홈 화면 둘러보기 (30초)

가입 후 홈에 도착하면:

[스크린샷 자리: 홈 화면 — 7개 메뉴 + 활동 피드]

좌측에 **7개의 상위 메뉴**가 보입니다:

1. **홈** — 지금 여기. 활동 피드와 주간 공정 미리보기
2. **일정** — 회사 전체 일정 (Phase 17에서 본격 신설)
3. **프로젝트** — 모든 프로젝트 목록
4. **지출관리** — 회계용 지출 내역
5. **AI 경리** — 자연어로 질문하는 회계 도구 (정식 출시 시 활성)
6. **팀관리** — 팀원·협력업체
7. **설정** — 회사 정보, 견적 기본비율, 템플릿

지금은 **3번 프로젝트** 메뉴로 이동합니다.

---

## 3단계 — 첫 프로젝트 만들기 (2분)

1. 좌측 메뉴에서 **"프로젝트"** 클릭
2. 우측 상단 **"+ 새 프로젝트"** 버튼 클릭
3. 다음을 입력합니다:

| 항목 | 입력 예시 |
|---|---|
| 프로젝트명 | `강남 30평 리모델링` |
| 고객명 | `김철수` |
| 현장 주소 | `서울 강남구 테헤란로 123, 405호` |
| 면적 (평) | `30` |
| 시작 예정일 | (오늘로부터 2주 후) |
| 종료 예정일 | (시작일로부터 6주 후) |

4. **"만들기"** 버튼 → 프로젝트 상세 페이지로 진입합니다.

> 💡 새 프로젝트가 만들어지면 회사 마감재 템플릿이 **자동 복사**됩니다. 즉, 빈 마감재 화면이 아니라 거실·주방·욕실 등 기본 SET이 이미 들어있는 상태로 시작합니다. 본인 회사 템플릿을 미리 다듬어두실수록 새 프로젝트가 빨라집니다.

[스크린샷 자리: 프로젝트 만들기 모달]

---

## 4단계 — 프로젝트 탭 한눈에 보기 (1분)

프로젝트에 진입하면 **10개의 탭**이 위에 가로로 나란히 있습니다. 자동으로 첫 번째 탭(**견적**)이 열립니다:

| # | 탭명 | 무엇을 하나 |
|---|---|---|
| 1 | **견적** (간편) | 공정별 1장짜리 빠른 견적 |
| 2 | 상세 견적 | 한국 표준 양식 갑지 + 18행 원가내역서 |
| 3 | 공정 일정 | 공정 단위 작업 간트 차트 |
| 4 | 마감재 | 자재·가전 통합 인라인 편집 |
| 5 | 마감재 (고급) | 폼 스키마 + 변경 이력 |
| 6 | 발주 | 거래처 발주 관리 |
| 7 | 체크리스트 | 공종별 점검 |
| 8 | 현장보고 | 일일 사진 보고 |
| 9 | 지출 | 현장별 영수증 |
| 10 | 메모 | Google Keep 스타일 카드 |

오늘 우리는 **1번 견적**과 **4번 마감재**, 두 곳만 손댑니다.

[스크린샷 자리: 프로젝트 탭 10개]

---

## 5단계 — 마감재 빠르게 채우기 (3분)

먼저 **4번 마감재** 탭으로 이동합니다.

이미 회사 템플릿에서 복사된 행들이 보일 겁니다. 빈 행들에 다음과 같이 입력하시면 됩니다 (예시):

| spaceGroup | itemName | kind | 단가 | 수량 | 단위 |
|---|---|---|---|---|---|
| 거실 | 강마루 — 메이플 색 | FINISH | 75,000 | 18 | m² |
| 거실 | 도배 — 실크 화이트 | FINISH | 12,000 | 50 | m² |
| 주방 | 싱크대 상판 — 인조대리석 | FINISH | 350,000 | 1 | 식 |
| 주방 | 식기세척기 LG DUE6BGL2E | APPLIANCE | 1,432,200 | 1 | 대 |
| 욕실 | 타일 — 600×300 화이트 | FINISH | 35,000 | 25 | m² |
| 욕실 | 양변기 — 일체형 | FINISH | 280,000 | 1 | 식 |

### 인라인 엑셀 편집 — 키보드만으로 충분합니다

행을 클릭하면 펼쳐져서 편집 가능합니다. 다음 키보드 단축키만 알면 빠릅니다:

| 키 | 동작 |
|---|---|
| `↑` `↓` | 위·아래 행으로 이동 |
| `←` `→` | 왼쪽·오른쪽 필드로 이동 (커서가 입력 끝/처음일 때) |
| `Tab` | 다음 필드. 마지막 필드에서 누르면 다음 행이 자동으로 펼쳐짐 |
| `Enter` | 다음 필드 (textarea 제외) |
| `Esc` | 펼친 행 닫기 |
| `F2` | 행 펼치기 |

> 💡 입력 후 **자동 저장**됩니다 (1초 뒤). 저장이 완료되면 행이 한 번 초록색으로 깜박입니다. **"저장" 버튼은 따로 누르지 않아도 됩니다.**

[스크린샷 자리: 마감재 인라인 편집 — 펼쳐진 행]

### 6상태 빠른 가이드

각 마감재 옆 상태칩 색이 다릅니다:

| 상태 | 의미 | 언제 |
|---|---|---|
| `UNDECIDED` | 미정 | 모델 결정 전 |
| `REVIEWING` | 검토 중 | 후보 좁혀짐 |
| `CONFIRMED` | 확정 | **견적·발주 모두 신뢰 가능** |
| `CHANGED` | 변경됨 | 1차 견적 후 변경된 항목 |
| `REUSED` | 재사용 | 기존 그대로 |
| `NOT_APPLICABLE` | 미적용 | 견적에서 제외 |

오늘은 일단 모든 항목을 `CONFIRMED`로 두고 진행합니다.

## 6단계 — 간편 견적 작성 (2분)

이제 **1번 견적** 탭으로 돌아갑니다.

1. **"+ 새 견적 만들기"** 버튼 클릭
2. 자동으로 빈 견적이 만들어집니다. 다음 행들을 입력합니다:

| 품명 | 규격 | 수량 | 단위 | 단가 | 비고 |
|---|---|---|---|---|---|
| 철거 공사 | 30평 일괄 | 1 | 식 | 2,500,000 | 폐기물 처리 포함 |
| 목공 / 가벽 | 거실+주방+안방 | 1 | 식 | 4,800,000 | |
| 도배 공사 | 실크 벽지 | 50 | m² | 35,000 | 자재비 포함 |
| 바닥 공사 | 강마루 메이플 | 18 | m² | 95,000 | 자재+시공 |
| 욕실 공사 | 공용 1, 안방 1 | 2 | 식 | 1,800,000 | 타일·양변기·세면대 |
| 주방 공사 | 싱크대·후드·식세기 | 1 | 식 | 5,200,000 | 가전 포함 |
| 전기 공사 | 30평 일괄 | 1 | 식 | 1,200,000 | LED 등기구 포함 |

### 견적 옵션 (좌측 또는 상단 패널)

| 옵션 | 입력 |
|---|---|
| 디자인 및 감리비 | `10` (%) — 기본값 |
| 단수조정 | `0` 또는 적당히 |
| 부가세 | `10` (%) — 클라이언트가 사업자가 아닌 일반 개인이면 보통 별도 안내라 `0`도 OK |
| 양식 | `classic` 또는 `modern` |

### 자동 계산되는 합계

표 아래 자동으로:

```
공정 합계      :   ?,???,???
디자인비 (10%) :     ???,???
단수조정       :          0
공급가액       :  ??,???,???
부가세 (10%)   :   ?,???,???
─────────────────────────────
총합계         :  ??,???,???
```

[스크린샷 자리: 견적 화면 — 표 + 합계 패널]

## 7단계 — PDF로 출력해서 클라이언트에게 보내기 (1분)

1. 견적 우측 상단 **"PDF 인쇄"** 또는 비슷한 버튼 클릭
2. 인쇄 미리보기가 뜹니다 (PrintModal)
3. 브라우저 인쇄 → **"PDF로 저장"** 선택
4. 파일명 예: `강남30평_김철수_견적_v1.pdf`
5. 카카오톡·이메일 등으로 클라이언트에게 전송

> ⚠️ **클라이언트는 Suplex에 들어올 수 없습니다.** PDF만 드립니다. 토큰 링크나 클라이언트 로그인 같은 기능은 없으며, 만들지 않습니다 — 회사 영업 정보 보호를 위함입니다. 자세한 이유는 [glossary.md - 클라이언트 비접근 원칙](glossary.md#클라이언트-비접근-원칙-모든-챕터-공통-약속) 참고.

## 자주 막히는 곳 (Quick FAQ)

**Q. 가입했는데 마감재에 행이 너무 많아요. 다 써야 하나요?**
A. 아니요. 회사 템플릿에 기본 SET이 들어가있을 뿐입니다. 안 쓰는 행은 상태를 `NOT_APPLICABLE`로 두거나, 우클릭으로 삭제하시면 됩니다. 또는 [Playbook #2](playbooks/02-company-template.md) 따라 회사 템플릿 자체를 정리하세요.

**Q. 부가세를 0%로 두면 클라이언트가 안 받아들여요.**
A. 사업자 클라이언트라면 부가세 10%를 명시해주는 게 맞습니다. 일반 개인 클라이언트라면 "부가세 별도"라고 안내하시거나 10%를 처음부터 포함시키시면 됩니다. 회사 견적 표준 비율은 **설정 → 견적 기본비율**에서 회사 단위로 미리 지정하실 수 있습니다.

**Q. PDF가 너무 길게 나와요 / 짧게 나와요.**
A. 양식을 `classic` ↔ `modern`으로 토글해보세요. classic은 빽빽하고 정통, modern은 여백 넓은 깔끔한 스타일입니다. 회사 양식이 정해져 있다면 [Playbook #2](playbooks/02-company-template.md)의 양식 설정 부분을 참고하세요.

**Q. 견적 보내고 나서 단가를 바꿔야 해요.**
A. 1차 견적은 그대로 두시고 **새 견적(2차)을 만드세요**. Suplex는 "변경 = 견적 2차" 원칙입니다. 1차를 수정하면 분쟁 시 증거가 망가집니다. 자세한 흐름은 [Playbook #3](playbooks/03-mid-project-change.md) 참고.

**Q. 다른 직원도 같은 프로젝트를 봐야 해요.**
A. **설정 → 팀관리**에서 팀원을 초대하세요. 임시 비밀번호가 발급되며 그 직원이 받아서 로그인 후 본인 비번으로 변경하면 됩니다. 자세한 절차는 [Playbook #6](playbooks/06-team-onboarding.md) 참고.

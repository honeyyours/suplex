# 챕터 9. 체크리스트

> 이 챕터를 읽고 나면 — 공종별 점검 항목을 등록하고, 시공 단계마다 체크하며, 누락된 작업을 한눈에 확인할 수 있게 됩니다.

---

## 9.1 체크리스트란

공사 진행 중 빠뜨리기 쉬운 점검 항목을 미리 등록해두고 단계별로 ✓ 체크하는 도구. 인테리어 업계에서 **하자 보수 비용을 가장 효과적으로 줄이는 도구**입니다.

---

## 9.2 항목 등록

프로젝트 → **체크리스트** 탭 (7번 탭).

### 공종별 그룹

| 공종 | 권장 점검 항목 |
|---|---|
| **철거** | 폐기물 분리 / 양생 처리 / 인접 호수 양해 |
| **목공** | 가벽 수직·수평 / 마감 단부 처리 / 단열재 누락 |
| **전기** | 콘센트 위치 / 콘센트 수량 / 인터넷 단자 / TV 안테나 / 누전차단기 |
| **타일** | 줄눈 일정 / 모서리 마감 / 시멘트 잔여물 / 배수구 기울기 |
| **도배** | 이음새 / 모서리 / 콘센트·스위치 주변 / 광원 반사 |
| **바닥** | 단차 / 걸레받이 / 문턱 / 마루 결 방향 |
| **수전** | 누수 검사 / 수전 흔들림 / 배수 속도 |
| **가구·가전** | 수평 / 작동 시험 / 매뉴얼·보증서 인계 |
| **청소·인계** | 입주 청소 / 키 인계 / 사용 설명 / 하자보수 안내 |

### 항목별 입력

| 필드 | 예시 |
|---|---|
| 공종 | 전기 |
| 항목명 | "거실 콘센트 8구 추가" |
| 점검자 | 회사 직원 또는 협력업체 |
| 예정일 | 2026-05-25 |
| 상태 | 미점검 / 점검중 / 완료 / 불합격 |
| 비고 | "재시공 필요" 등 |

---

## 9.3 회사 표준 체크리스트 등록

설정 → 체크리스트 템플릿에 회사 평소 점검 항목을 등록해두면, 새 프로젝트마다 자동 복사됩니다.

> 💡 회사 도입 첫 달의 가장 가치 있는 작업: **모든 공종의 점검 항목을 회사 표준으로 정리하기.** 한 번 만들어두면 모든 프로젝트가 자동 혜택.

---

## 9.4 모바일 활용

체크리스트는 모바일 화면에서도 잘 동작합니다. 현장에서 점검하면서 즉시 ✓ 체크 가능.

# 챕터 13. 지출 관리

> 이 챕터를 읽고 나면 — 프로젝트별 지출을 입력하고, 견적 대비 마진을 확인할 수 있게 됩니다.

---

## 13.1 지출관리의 두 위치

| 위치 | 단위 | 용도 |
|---|---|---|
| **상위 메뉴 → 지출관리** | 회사 전체 | 회계용 (P&L, 재무제표) |
| **프로젝트 → 지출 탭** (9번 탭) | 한 프로젝트 | 현장 마진 분석 |

두 곳 데이터는 연결되어 있어 어느 쪽에서 입력해도 양쪽에 반영됩니다.

---

## 13.2 지출 입력

**+ 새 지출** → 입력:

| 필드 | 예시 |
|---|---|
| 거래처 | 협력업체 마스터에서 선택 |
| 계정과목 | 자재비 / 노무비 / 경비 / 관리비 등 |
| 금액 | 원 단위 |
| 부가세 | 자동 계산 또는 직접 입력 |
| 결제수단 | 카드 / 계좌이체 / 현금 |
| 결제일 | 실제 결제 날짜 |
| 프로젝트 연결 | 어느 현장 (선택, 비연결 가능) |
| 비고 | |

[스크린샷 자리: 지출 입력 화면]

---

## 13.3 CSV 임포트 / 엑스포트

### 임포트

은행·카드사에서 다운받은 거래내역을 CSV로 업로드 가능. 자동으로 계정과목 추정 (회사 거래 패턴 학습).

### 엑스포트

| 용도 | 형식 |
|---|---|
| 회계사 전달용 | CSV (계정과목·금액·날짜) |
| 부가세 신고 | 매입세금계산서 양식 |
| 손익계산서 | P&L 보고서 |

---

## 13.4 P&L (손익) 뷰

상위 메뉴 → 지출관리 → P&L 탭.

| 분기·월 단위 | 매출 / 매입 / 인건비 / 경비 / 영업이익 |

회사 전체 회계 흐름을 한눈에. 다만 **본격적인 회계 기능은 회계 ERP에 위탁** 권장 — Suplex는 인테리어 운영에 필요한 만큼만 다룹니다.

---

## 13.5 프로젝트별 마진 분석

프로젝트 → 지출 탭에서:

```
견적 합계      :  36,000,000
지출 합계      :  28,500,000
─────────────────────────────
마진           :   7,500,000  (20.8%)
```

자동 계산. 마진율이 회사 목표(예: 25%) 미달 시 경고.

### Suplex 코드 탐색 결과 반영
- 프로젝트 탭은 5개가 아닌 **10개** (간편견적/상세견적/공정일정/마감재/마감재고급/발주/체크리스트/현장보고/지출/메모)
- 마감재 상태는 4가지가 아닌 **6가지** (UNDECIDED/REVIEWING/CONFIRMED/CHANGED/REUSED/NOT_APPLICABLE)
- 부가세 기본값은 10%가 아닌 **0%** (별도 안내), 사용자 설정
- 발주 시스템은 이미 구현되어 있음 (메모리에 "분리 대기"로 기록되어 있던 것 정정)
- 간편견적의 "직전 차수와 비교" 기능은 AI 자동 요약 → footerNotes 자동 주입

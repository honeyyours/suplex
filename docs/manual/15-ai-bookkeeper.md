# 챕터 14. AI 경리

> ⚠️ **본 기능은 정식 출시 전 보류 상태입니다 (2026-04-26 시점).** 정식 출시 시 구독 등급 차등용으로 활성화될 예정. 본 챕터는 기능 활성화 시 즉시 채워질 자리입니다.

---

## 14.1 AI 경리란

자연어로 회계·운영 질문을 던지면 답하는 도구. Suplex의 회계 데이터(지출관리·견적·발주)를 기반으로 답변.

### 예상 질문 시나리오

```
질문: "이번 달 가장 큰 매입처가 어디야?"
답변: "이번 달(2026-04) 매입 1위는 XYZ타일 (1,250만원, 전체 매입의 38%)입니다.
       2위는 ABC도배 (820만원), 3위는 LG가전 (640만원)입니다."

질문: "지난 분기 평균 마진율은?"
답변: "2026 Q1 매출 1.8억, 매입+인건비 합 1.42억. 평균 마진율 약 21%입니다.
       30평 이하 프로젝트는 평균 25%, 그 이상은 19%로 차이가 있습니다."

질문: "강남 래미안 304-1502 지금 마진은?"
답변: "현재 견적 36,400,000원 / 지출 누적 18,200,000원.
       견적 대비 50% 진행 중이며 마감재 발주가 70% 완료된 상태입니다.
       예상 마진 약 22%."
```

## 14.2 무엇이 가능할 예정

| 영역 | 질문 예시 |
|---|---|
| 매입·매출 | "이번 달 매출", "지난 분기 매입처별 합" |
| 마진 분석 | "프로젝트별 마진율", "공종별 평균 단가" |
| 거래처 인사이트 | "신뢰도 높은 거래처 TOP 5" |
| 일정 | "다음 주 마감 임박 공정", "거래처 일정 충돌" |
| 비교 | "올해와 작년 같은 분기 비교" |

## 14.3 기능 출시 전 알림

이 챕터는 정식 출시 후 다음 내용이 채워집니다:

- AI 사용법 (질문 입력, 결과 확인)
- 답변 정확도와 한계
- 데이터 보안 (민감 정보 처리)
- 구독 등급별 사용 한도
- 자주 묻는 질문 (FAQ 추가)

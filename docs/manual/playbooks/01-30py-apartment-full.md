# Playbook #1 — 30평 아파트 전면 리모델링

**첫 미팅에서 PDF 견적 발송까지, 그리고 1차 변경 후 2차 견적까지의 실전 시나리오**

> 이 Playbook을 읽고 따라하시면 — 실제 현장에서 가장 자주 만나는 패턴(첫 견적 → 클라이언트 변경 요청 → 2차 견적)을 끝까지 처리할 수 있게 됩니다.

소요 시간: 처음 따라하실 때 약 30~45분, 익숙해지면 같은 시나리오를 10분 안에 끝낼 수 있습니다.

---

## 시나리오 배경

```
회사: ABC인테리어 (직원 4명, 강남 일대 아파트 리모델링 전문)
사용자(견적 담당): 김미영 사원
프로젝트: 강남 30평 리모델링 — "강남 래미안" 304동 1502호
클라이언트: 박상철 (40대 남성, 가족 4인, 입주 6주 후 예정)
첫 미팅: 2026년 5월 10일 토요일 오후 2시, 현장에서 진행
```

이 한 시나리오 안에 다음 모든 것이 등장합니다:

| 단계 | 사용 기능 |
|---|---|
| ① 미팅 직후 프로젝트 등록 | 프로젝트 생성, 회사 템플릿 자동 복사 |
| ② 마감재 입력 | 마감재 (간편) 탭, 인라인 편집, 6상태 |
| ③ 가전 입력 | APPLIANCE kind, 가전 규격 마스터 활용 |
| ④ 간편 견적 작성 | 견적 탭, 디자인비/단수조정/부가세 |
| ⑤ PDF 발송 | PrintModal, classic 양식 |
| ⑥ 클라이언트 변경 요청 | 마감재 변경 (CHANGED 상태) |
| ⑦ 2차 견적 만들기 | 1차 SUPERSEDED, 2차 새로 생성 |
| ⑧ 직전 차수 비교 | AI 자동 요약 → footerNotes 자동 주입 |
| ⑨ 2차 PDF 발송 | "1차 대비 변경 요약" 한 줄 첨부 |

---

## 단계 ① — 미팅 직후 프로젝트 등록 (5분)

미팅에서 받은 정보 정리부터 시작합니다.

**미팅에서 알아낸 것**:
- 30평 아파트 (전용 84㎡)
- 거실 + 안방 + 작은방 2개 + 주방 + 욕실 2개 + 베란다 확장 1
- 6주 안에 입주 예정 → 공기 빠듯
- 클라이언트 취향: 따뜻한 톤, 메이플 강마루, 화이트 인테리어
- 가전: 식기세척기 빌트인 + 인덕션 + 후드 새로 설치 희망
- 예산: 약 5,000만 원 (상한선 5,500만 원)

### 입력 절차

1. 좌측 메뉴 → **프로젝트** → **+ 새 프로젝트**
2. 다음 정보를 입력합니다:

| 항목 | 입력값 |
|---|---|
| 프로젝트명 | `강남 래미안 304-1502 리모델링` |
| 고객명 | `박상철` |
| 연락처 | `010-9876-5432` |
| 현장 주소 | `서울 강남구 도곡동 123, 강남 래미안 304동 1502호` |
| 면적 (평) | `30` |
| 시작 예정일 | `2026-05-15` |
| 종료 예정일 | `2026-06-26` |
| 예상 견적 | `50,000,000` (참고용, 실제 견적과 별개) |

3. **만들기** → 프로젝트 상세로 진입

> 💡 **자동으로 복사된 마감재 템플릿**: 회사 설정에 등록해둔 기본 마감재 SET이 이 프로젝트에 자동으로 들어갑니다. ABC인테리어는 평소에 다음을 템플릿에 등록해뒀다고 가정합니다:
>
> - 거실: 강마루, 도배, 등기구 / 안방·작은방: 강마루, 도배, 붙박이장 / 주방: 싱크대 상판, 주방 가구, 후드 / 욕실: 타일, 양변기, 세면대, 샤워기, 거울 / 공통: 현관 중문, 창호 필름, 베란다 마감
>
> 즉, 프로젝트 만든 직후에도 마감재 탭이 빈 화면이 아니라 **30~50개 행이 미리 채워진 상태**입니다.

[스크린샷 자리: 새 프로젝트 만들기 직후 — 마감재 탭이 자동으로 채워져 있는 모습]

### 입력 노하우 — 키보드만으로 빠르게

위 표를 30개 이상 입력하셔야 합니다. 마우스만 쓰면 30분 걸립니다. 키보드만 쓰면 5분입니다.

## 단계 ③ — 간편 견적 1차 작성 (8분)

**견적** 탭(1번 탭)으로 이동.

`+ 새 견적 만들기` → 빈 견적이 만들어지고 표가 뜹니다.

박상철 고객용 견적은 다음과 같이 구성합니다:

| 품명 | 규격 | 수량 | 단위 | 단가 | 금액 | 비고 |
|---|---|---|---|---|---|---|
| 철거 공사 | 30평 일괄 | 1 | 식 | 2,500,000 | 2,500,000 | 폐기물 처리 포함 |
| 목공 + 가벽 | 거실+안방+작은방 | 1 | 식 | 4,800,000 | 4,800,000 | |
| 도배 공사 | 실크 벽지 (전체) | 132 | m² | 25,000 | 3,300,000 | 자재 + 시공 |
| 바닥 공사 | 강마루 메이플 (전체) | 45 | m² | 95,000 | 4,275,000 | 자재 + 시공 |
| 욕실 공사 | 공용 + 안방 | 2 | 식 | 1,800,000 | 3,600,000 | 타일·양변기·세면대·수전 |
| 주방 공사 | 싱크대 + 가전 | 1 | 식 | 6,800,000 | 6,800,000 | 식세기·인덕션·후드 포함 |
| 전기 공사 | 30평 일괄 | 1 | 식 | 1,200,000 | 1,200,000 | LED + 콘센트 추가 |
| 창호·필름 | 6창 + 중문 | 1 | 식 | 1,800,000 | 1,800,000 | |
| 베란다 확장 | 1실 | 1 | 식 | 1,500,000 | 1,500,000 | 단열재 + 도배 |
| 현장 정리 | 일괄 | 1 | 식 | 350,000 | 350,000 | |

**공정 합계**: 30,125,000원

### 견적 옵션 입력

상단 또는 측면 옵션 패널에:

| 항목 | 값 | 의미 |
|---|---|---|
| 디자인 및 감리비 | 10 (%) | 공정 합계의 10% |
| 단수조정 | -25,000 | 30,000,000원 단위 맞춤용 |
| 부가세 | 10 (%) | 일반 개인 고객이라도 사업자 거래 시 명시 |
| 양식 | classic | ABC인테리어 정통 양식 |

### 자동 계산 결과

```
공정 합계      :  30,125,000
디자인비 (10%) :   3,012,500
단수조정       :     -25,000
공급가액       :  33,112,500
부가세 (10%)   :   3,311,250
─────────────────────────────
총합계         :  36,423,750
```

> ⚠️ 클라이언트가 미팅 때 "5천만 원 정도 생각한다"고 하셨는데 견적이 3,640만 원으로 낮게 나왔습니다. 가구·가전 구입은 별도라는 점, 자재 등급 업그레이드 옵션 등을 PDF에 명시해두는 게 좋습니다 (footerNotes 활용).

### footerNotes (견적서 하단 안내문) 입력

견적 하단에는 PDF에 같이 출력될 안내문을 넣을 수 있습니다:

```
※ 본 견적서는 2026년 5월 10일 현장 미팅 기준으로 작성되었습니다.
※ 가구류(소파·식탁·침대) 및 가전 일부(TV·세탁기 등)는 별도 구매 사항입니다.
※ 클라이언트 직접 구매하신 자재는 견적에서 제외되어 있습니다.
※ 자재 등급 변경 시 단가가 변동될 수 있습니다.
※ 본 견적의 유효기간은 발행일로부터 30일입니다.
```

### 상태 전환

견적을 작성 완료하면 상태를 **DRAFT → SENT**로 전환합니다 (상단 또는 우측의 상태 버튼). 이렇게 해야 향후 "직전 차수 비교"의 기준점이 됩니다.

## 단계 ④ — PDF 출력 및 발송 (2분)

1. 견적 우측 상단 **PDF 인쇄** (또는 비슷한 라벨) 버튼 클릭
2. **PrintModal** 미리보기가 뜸
3. 미리보기에서 양식이 잘 보이는지 확인:
   - 회사 로고·정보 (상단)
   - 클라이언트 정보 (좌측)
   - 견적 표 (중앙)
   - 합계 (하단)
   - footerNotes (가장 아래)
4. **인쇄** → 대상 프린터 → **"PDF로 저장"** 선택
5. 파일명: `강남래미안304-1502_박상철_견적_v1_2026-05-10.pdf`
6. 카카오톡으로 박상철 고객에게 전송:

> 박상철님, 안녕하세요. ABC인테리어 김미영입니다.
> 오늘 미팅 잘 끝내주셔서 감사드립니다.
> 견적서 첨부해드립니다. 검토 후 편하실 때 연락 주시면 감사하겠습니다.
> 감사합니다.

[스크린샷 자리: PrintModal 미리보기]

## 단계 ⑤ — 사흘 뒤, 클라이언트 변경 요청 (현실 시뮬레이션)

3일 후, 박상철 고객으로부터 카카오톡:

> 김미영님, 견적서 잘 받아봤습니다. 두 가지만 부탁드릴게요:
>
> 1. 욕실 타일을 화이트 무광 대신 **그레이 무광**으로 바꾸고 싶어요. 가격은 비슷하면 좋을 것 같고요.
> 2. 강마루를 메이플에서 **오크 색**으로 변경해주세요. 와이프가 더 진한 톤을 선호한다고 합니다.
> 3. 그리고 거실등 펜던트는 직접 안 사고 추천해주신 모델로 부탁드립니다.

이 변경을 처리하는 두 가지 방법이 있습니다:

| 방법 | 언제 |
|---|---|
| **A. 1차 견적 직접 수정** | ❌ 절대 권장하지 않음. 분쟁 시 증거 망가짐 |
| **B. 1차는 그대로 두고 2차 견적 새로 생성** | ✅ Suplex 표준 |

**Suplex 원칙: 변경 = 견적 2차.** 1차는 SUPERSEDED 상태가 되고, 2차가 새로 만들어집니다. 둘 다 영구 보관됩니다.

## 단계 ⑥ — 마감재 변경 (3분)

**마감재** 탭으로 이동.

해당 행들을 찾아 다음과 같이 수정합니다:

### 변경 1: 욕실 타일

| 변경 전 | 변경 후 |
|---|---|
| spaceGroup: 공용욕실 | spaceGroup: 공용욕실 |
| itemName: **벽 타일 — 600×300 화이트 무광** | itemName: **벽 타일 — 600×300 그레이 무광** |
| 단가: 35,000 | 단가: 36,000 (그레이는 약간 비쌈) |
| 상태: CONFIRMED | **상태: CHANGED** |

> 💡 **상태를 `CHANGED`로 바꾸시는 것이 핵심**입니다. 이렇게 해두면 2차 견적에서 어떤 항목이 바뀐 것인지 시각적으로 강조됩니다.

같은 방식으로 **안방욕실 벽 타일도 CHANGED로 변경**.

### 변경 2: 강마루 색상

거실·안방·작은방-1·작은방-2의 강마루 4개 행 모두:

| 변경 전 | 변경 후 |
|---|---|
| itemName: **강마루 — 메이플 색 (NF-201)** | itemName: **강마루 — 오크 색 (NF-305)** |
| 단가: 75,000 | 단가: 78,000 (오크는 약간 비쌈) |
| 상태: CONFIRMED | **상태: CHANGED** |

### 변경 3: 거실등 펜던트

| 변경 전 | 변경 후 |
|---|---|
| 비고: 클라이언트 모델 직접 구매 | 비고: ABC인테리어 추천 모델로 변경 |
| 단가: 280,000 (참고용, 실제는 직접 구매여서 견적 외) | 단가: 320,000 (회사 매입가) |
| 수량: 1 (별도) | 수량: 1 |
| 상태: CONFIRMED | **상태: CHANGED** |

### 자동으로 일어나는 일 — MaterialHistory

마감재 변경 사항은 자동으로 **MaterialHistory 테이블**에 기록됩니다:
- 누가 변경했는가 (김미영 사원)
- 언제 변경했는가 (2026-05-13 14:30)
- 어떤 필드가 (itemName, unitPrice 등)
- 어떤 값에서 → 어떤 값으로

이 이력은 **마감재(고급) 탭** 또는 **변동로그**에서 조회 가능합니다. 분쟁 시 결정적 증거가 됩니다.

[스크린샷 자리: 마감재 탭 — CHANGED 상태로 표시된 행들]

## 단계 ⑦ — 2차 견적 만들기 (5분)

**견적** 탭으로 돌아갑니다.

### 1차 견적 상태 잠그기

먼저 1차 견적의 상태를 **SENT → SUPERSEDED**로 변경합니다 (또는 2차 생성 시 자동으로 전환되는 경우도 있음 — 화면 안내에 따라).

> 💡 SUPERSEDED는 "이 견적은 더 이상 유효하지 않음, 새로 차수가 나옴"이라는 표시입니다. 영구 보관은 되지만 클라이언트 협의의 기준점이 아니라는 뜻.

### 2차 견적 생성

`+ 새 견적 만들기` (또는 `1차 복제` 같은 기능이 있다면 그쪽이 더 빠름).

1차 내용을 복사한 후 변경된 부분만 수정:

| 품명 | 1차 단가 | 2차 단가 | 비고 |
|---|---|---|---|
| 도배 공사 | 25,000/m² | 25,000/m² | (변동 없음) |
| **바닥 공사** | **95,000/m²** | **98,000/m²** | 강마루 오크 단가 인상분 반영 |
| **욕실 공사** | **1,800,000/식 × 2** | **1,830,000/식 × 2** | 그레이 타일 단가 인상분 |
| 거실등 펜던트 (별도 라인) | (없었음) | **320,000/개 × 1** | 신규 추가 |

### 자동 계산 결과 (2차)

```
공정 합계      :  30,560,000  (+ 435,000)
디자인비 (10%) :   3,056,000  (+ 43,500)
단수조정       :     -16,000  (보정)
공급가액       :  33,600,000  (+ 487,500)
부가세 (10%)   :   3,360,000  (+ 48,750)
─────────────────────────────
총합계         :  36,960,000  (+ 536,250)
```

## 단계 ⑧ — 직전 차수와 비교 (AI 자동 요약) (1분)

여기가 Suplex의 진짜 매력입니다.

2차 견적 화면에서 **"직전 차수와 비교"** 버튼 클릭.

### 자동으로 일어나는 일

1. Suplex가 1차 견적과 2차 견적을 비교합니다.
2. AI가 차이를 한국어로 자동 요약합니다.
3. 요약문이 2차 견적의 **footerNotes(하단 안내문)에 자동으로 추가**됩니다.

자동 추가되는 footerNotes 예시:

```
※ 본 견적은 2차(변경) 견적입니다. 1차 대비 변경 사항은 다음과 같습니다:
  - 강마루 색상 변경: 메이플 → 오크 (전체)
  - 욕실 타일 색상 변경: 화이트 무광 → 그레이 무광 (공용·안방)
  - 거실등 펜던트 추가 (320,000원)
  - 합계 차이: +536,250원

※ 본 견적서는 2026년 5월 14일 변경 요청 기준으로 작성되었습니다.
※ 1차 견적(2026-05-10 발행)은 SUPERSEDED 처리되었으며, 본 2차 견적이 유효합니다.
```

> 💡 클라이언트가 PDF만 받으면 어떤 변경이 일어났는지 한눈에 파악합니다. "왜 53만 원이 늘었나요?" 같은 질문이 안 나옵니다.

[스크린샷 자리: 직전 차수 비교 결과 — 자동 요약 + footerNotes 자동 입력]

## 단계 ⑨ — 2차 PDF 발송 (2분)

1차와 동일하게 **PDF 인쇄** → **PDF로 저장**:
- 파일명: `강남래미안304-1502_박상철_견적_v2_2026-05-14.pdf`

카카오톡 안내:

> 박상철님, 요청하신 변경사항 반영해서 2차 견적서 보내드립니다.
> 1차 대비 주요 변경 사항은 견적서 하단에 정리해뒀습니다.
> 검토 후 편하실 때 연락 주시면 감사하겠습니다.

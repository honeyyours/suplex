# Playbook #7 — 영수증 한 장에서 정산까지

**현장에서 결제한 영수증을 즉시 입력하고 → 월말 회계 마감까지의 흐름**

> 이 Playbook을 읽고 따라하시면 — 매일 발생하는 지출을 즉시 기록하고, 월말에 회계사에게 정확한 자료를 넘길 수 있게 됩니다.

소요 시간: 매 영수증 1분 입력 + 월말 정리 1시간

---

## 시나리오

```
ABC인테리어 김미영, 5/15 (화) 오후
강남 래미안 304-1502 현장 들렀다가 자재상에 들름.
- 양변기 부속 (W샵): 카드 결제 95,000원
- 콘센트 추가 자재 (전기상): 현금 결제 28,000원
- 점심 식대 (현장 직원 5명): 카드 50,000원

김미영: "이런 자잘한 영수증을 매번 챙겨놔야 하는데..."
```

---

## 단계 ① — 영수증 사진 즉시 촬영 (그 자리에서, 30초)

영수증 받자마자:
1. 휴대폰 카메라 → 영수증 촬영 (밝은 곳에서 평면)
2. 글자 잘 보이는지 확인 (안 보이면 재촬영)
3. 그 자리에서 Suplex 앱/웹으로 입력 (또는 사진 갤러리에 모아두고 사무실에서)

> 💡 **그 자리에서 입력**이 가장 좋습니다. 모아두면 나중에 "이게 어디 영수증이지?" 헷갈립니다.

---

## 단계 ② — Suplex에 입력 (1분)

좌측 메뉴 → **지출관리** → **+ 새 지출** (또는 프로젝트 안에서 지출 탭).

### 양변기 부속

| 필드 | 입력 |
|---|---|
| 거래처 | W샵 (협력업체 마스터에서 선택, 없으면 "기타") |
| 계정과목 | 자재비 |
| 금액 | 95,000 |
| 부가세 | 자동 계산 (보통 10% = 8,636 / 86,364) |
| 결제수단 | 카드 (회사 카드) |
| 결제일 | 2026-05-15 |
| 증빙 사진 | (방금 찍은 영수증) |
| 프로젝트 연결 | 강남 래미안 304-1502 |
| 비고 | 안방 욕실 양변기 추가 부속 |

### 전기상 자재

| 필드 | 입력 |
|---|---|
| 거래처 | 전기상 (마스터에 없으면 "현장 자재상") |
| 계정과목 | 자재비 |
| 금액 | 28,000 |
| 부가세 | 0 (간이영수증, VAT 없음) |
| 결제수단 | 현금 |
| 결제일 | 2026-05-15 |
| 증빙 사진 | 영수증 |
| 프로젝트 연결 | 강남 래미안 304-1502 |

### 점심 식대

| 필드 | 입력 |
|---|---|
| 거래처 | (식당 이름) |
| 계정과목 | 복리후생비 |
| 금액 | 50,000 |
| 부가세 | 자동 (10% = 4,545) |
| 결제수단 | 카드 |
| 결제일 | 2026-05-15 |
| 증빙 사진 | 영수증 |
| 프로젝트 연결 | 강남 래미안 304-1502 (현장 직원 점심) |

> 💡 **식대는 프로젝트 연결**: 자체 직원 식대라도 어느 현장에서 발생했는지 기록하면 프로젝트별 마진 분석이 정확해집니다.

---

## 단계 ③ — 매주 금요일 점검 (10분)

매주 금요일 오후 5시:
1. 좌측 메뉴 → 지출관리
2. 이번 주 (월~금) 필터
3. 누락된 영수증 없는지 점검 — 카드 명세서와 대조

> 💡 **카드사 거래내역 CSV** 다운받아서 Suplex에 임포트하시면 자동으로 모든 카드 결제가 들어옵니다 (계정과목은 본인이 분류).

---

## 단계 ④ — 월말 회계 마감 (1시간)

매월 말일 또는 익월 1주 안에:

### 4-1. 모든 지출 점검

이번 달 지출 리스트 → 각 항목:
- 증빙 사진 첨부됐는가
- 계정과목 맞는가 (자재비/노무비/경비/복리후생비 등 정확)
- 부가세 계산 맞는가
- 프로젝트 연결됐는가 (마진 분석용)

### 4-2. CSV 엑스포트 → 회계사

지출관리 → 엑스포트 → CSV 또는 매입세금계산서 양식.

회계사에게 카카오톡 또는 이메일:

> "ABC인테리어 김갑수입니다.
> 2026년 5월 매입 자료 첨부합니다.
> CSV 형식으로 정리되어 있습니다.
> 추가 요청사항 있으시면 말씀주세요."

### 4-3. P&L 확인

지출관리 → P&L 탭:

```
2026년 5월
 매출 (입금 합계)        : 125,000,000
 매입 (지출 합계)        :  98,500,000
 ─────────────────────────────
 영업이익               :  26,500,000  (21.2%)
 
 프로젝트별 마진율:
   강남 래미안 304-1502  : 22.1%
   도곡 자이 1102호      : 19.8%
   ...
```

---

## 단계 ⑤ — 분기 결산 (분기 마지막 달)

3개월에 한 번:
1. P&L 분기 비교 (지난 분기 대비)
2. 큰 매입처 TOP 10 (어디에 돈을 가장 많이 쓰는가)
3. 마진율 낮은 프로젝트 분석 (왜 마진이 낮았나)
4. 회사 템플릿 단가 갱신 (필요 시)

---

## 자주 막히는 곳

**Q. 영수증을 자주 잃어버립니다.**
A. 영수증을 받자마자 **30초 안에 사진 찍고 Suplex 입력**이 답입니다. 종이는 잃어버려도 사진이 남으면 OK.

**Q. 현금 영수증인데 사업자번호 없으면 어떻게 하나요?**
A. 한국 부가세법상 간이영수증·신용카드 영수증·현금영수증이 매입세금계산서를 대체합니다. Suplex에 사진만 첨부하면 회계사가 처리.

**Q. 거래처가 협력업체 마스터에 없는데요.**
A. **마스터 등록 안 하고 "거래처: 기타 — XXX 자재상"** 같이 텍스트로 입력해도 됩니다. 자주 거래하게 되면 그때 마스터에 등록.

**Q. 부가세 계산이 헷갈립니다.**
A. Suplex가 자동 계산합니다. 직접 입력 안 하셔도 OK. 다만 간이영수증(VAT 없는 경우)은 부가세 0으로 두세요.

**Q. 회사 카드와 개인 카드 섞여 있어요.**
A. 결제수단 필드로 구분하시면 됩니다. 개인 카드로 결제한 회사 비용은 "개인 카드 매출 환급" 처리 (회계사와 협의).

| 작업 | 효과 |
|---|---|
| 영수증 즉시 사진 + 입력 | 누락 0%, 회계사 자료 1주 → 1일 |
| 매주 금요일 점검 | 월말 폭주 방지 |
| 프로젝트 연결 | 정확한 마진 분석 가능 |
| CSV 자동 엑스포트 | 회계사 비용 절감 |
| P&L 자동 계산 | 사장님 의사결정 빨라짐 |

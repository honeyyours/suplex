# Playbook #4 — 마감재 단가 인상 일괄 반영

**자재 도매가 인상 시 모든 진행 중 프로젝트 영향 분석 + 회사 템플릿 갱신**

> 이 Playbook을 읽고 따라하시면 — 자재 단가가 변동됐을 때 어느 프로젝트가 영향받는지 빠르게 파악하고, 회사 표준 단가를 한 번에 갱신할 수 있게 됩니다.

소요 시간: 30분~1시간

---

## 시나리오

```
2026-05-30: LG 가전 가격 인상 발표
- 식기세척기 모델 평균 5% 인상
- 인덕션 모델 평균 8% 인상
- 6/10부터 적용

ABC인테리어 김미영: "어... 진행 중인 프로젝트가 5개인데
어디까지 영향받는 거지? 새 견적도 다음 주부터는 새 단가로
나가야 할 텐데."
```

---

## 단계 ① — 영향받는 프로젝트 식별 (10분)

좌측 메뉴 → **프로젝트** → 필터: **상태 = ACTIVE**.

각 프로젝트에서 **마감재** 탭 → APPLIANCE kind 필터 → LG 식기세척기·인덕션 검색.

영향받는 프로젝트 정리:

| 프로젝트 | LG 식세기 | LG 인덕션 | 발주 상태 | 영향 |
|---|---|---|---|---|
| 강남 래미안 304-1502 | DUE6BGL2E (CONFIRMED) | BEY3GST (CONFIRMED) | 미발주 | ⚠️ 단가 갱신 필요 |
| 도곡 자이 1102호 | (없음) | BEY5GST (CONFIRMED) | 발주 완료 | ✅ 발주가 잠금 — 영향 없음 |
| 청담 더샵 805호 | DUE6BGE (REVIEWING) | (없음) | 미결정 | ⚠️ 새 단가 반영 |
| 서초 우성 502호 | (없음) | (없음) | - | ✅ 영향 없음 |
| 잠실 엘스 1503호 | DEE6BGE (CONFIRMED) | BEY3GST (CONFIRMED) | 미발주 | ⚠️ 단가 갱신 필요 |

→ **3개 프로젝트가 단가 갱신 필요**, 1개는 발주 완료라 영향 없음.

---

## 단계 ② — 회사 템플릿(가전 규격 마스터) 단가 갱신 (10분)

**설정 → 가전 규격 마스터** (또는 **마감재 템플릿**).

해당 모델들의 단가를 갱신:

| 모델 | 이전 단가 | 새 단가 | 변동 |
|---|---|---|---|
| LG DUE6BGL2E | 1,432,200 | 1,503,800 | +5.0% |
| LG DEE6BGE | 1,250,000 | 1,312,500 | +5.0% |
| LG DUE6BGE | 1,380,000 | 1,449,000 | +5.0% |
| LG BEY3GST | 1,180,000 | 1,274,400 | +8.0% |
| LG BEY5GST | 1,580,000 | 1,706,400 | +8.0% |

> ⚠️ **회사 템플릿 단가는 새 프로젝트의 기본값에만 영향**입니다. 이미 만들어진 프로젝트의 마감재 행은 자동으로 안 바뀝니다 (그래야 안전 — 진행 중 프로젝트 단가가 자동으로 바뀌면 견적 일관성 깨짐).

---

## 단계 ③ — 영향받는 프로젝트 수동 갱신 (15분)

영향받는 3개 프로젝트 각각:

### 프로젝트 1: 강남 래미안 304-1502

마감재 탭:
- LG DUE6BGL2E 행: 단가 1,432,200 → 1,503,800 (+71,600), status = CHANGED
- LG BEY3GST 행: 단가 1,180,000 → 1,274,400 (+94,400), status = CHANGED

견적 탭:
- 1차 견적 SENT/ACCEPTED 상태면 → SUPERSEDED 처리, 2차 견적 새로 만들기
- 1차 견적 DRAFT 상태면 → 그 자리에서 라인 단가만 수정 (안전, 아직 발송 안 했으니)

### 프로젝트 2: 청담 더샵 805호

식기세척기는 REVIEWING 상태였으니, CONFIRMED로 갈 때 새 단가로 입력.

### 프로젝트 3: 잠실 엘스 1503호

(프로젝트 1과 동일 절차)

---

## 단계 ④ — 클라이언트 응대 (각 프로젝트별)

CHANGED 처리한 프로젝트의 클라이언트에게 안내:

> [클라이언트명]님, 안녕하세요. ABC인테리어 김미영입니다.
> 5/30 LG 가전 가격 인상 발표가 있었습니다.
> [모델명] 단가가 [변동액] 인상되어, 견적에 반영해 2차 견적 보내드리겠습니다.
> 하루 내 회신드리겠습니다.

→ 2차 견적 작성 + AI 자동 요약 → PDF 발송.

---

## 단계 ⑤ — 회사 견적 라인 템플릿도 갱신 여부 검토

가전 단가 변동만이면 **견적 라인 템플릿(공정 단위)은 영향 없을 수도**. 만약 "주방 공사 (식세기·인덕션·후드 포함)" 같은 식 단위 라인이 있다면 그것도 같이 갱신.

---

## 단계 ⑥ — 변경 이력 메모 (5분)

**프로젝트 → 메모 탭**에 단가 변동 메모 카드 추가 (해당 프로젝트들):

```
제목: LG 가전 단가 인상 (2026-05-30)
태그: 자재 단가, 외부 변동

LG 가전 가격 인상 (6/10부터 적용)
- 식기세척기 +5%
- 인덕션 +8%

본 프로젝트 영향:
- DUE6BGL2E: 1,432,200 → 1,503,800
- BEY3GST: 1,180,000 → 1,274,400

5/30 발표 → 5/31 클라이언트에게 2차 견적 발송 → 6/1 합의 완료
```

---

## 끝났습니다 — 이 시나리오의 가치

| 작업 | 효과 |
|---|---|
| 회사 템플릿 단가 갱신 | 이후 모든 새 프로젝트 자동 새 단가 |
| 진행 중 프로젝트 영향 식별 | 5개 중 3개만 작업 → 시간 절약 |
| CHANGED 상태 활용 | 어느 항목이 단가 변동인지 추적 |
| AI 자동 요약 | 각 클라이언트에게 변경 사유 명확 전달 |
| 메모 카드에 외부 사유 기록 | 분쟁 시 "회사 일방 인상 아님" 증거 |

---

## 자주 막히는 곳

**Q. 모든 프로젝트에 일괄 단가 적용은 안 되나요?**
A. 의도적으로 안 됩니다. **진행 중 프로젝트의 단가가 자동으로 바뀌면 견적 일관성이 깨집니다.** 사용자가 의식적으로 결정 → 클라이언트에게 안내하는 흐름이 분쟁 예방에 안전합니다.

**Q. 클라이언트가 "1차 단가로 가달라"고 하면?**
A. 발주 완료 전이면 회사 손해 발생, 발주 완료 후이면 거래처에 이미 새 단가로 매입했으니 이중 부담. 협상 카드:
- (1) 다른 자재에서 단가 조정 (예: 다른 라인을 회사 부담으로 -10만원)
- (2) 디자인비 % 협상
- (3) 정중히 거절 ("죄송하지만 거래처 단가 변동이라 어쩔 수 없습니다")

**Q. 가전 규격 마스터에 없는 모델은 어떻게 갱신?**
A. 직접 마감재 탭에서 수동 입력했던 모델은 회사 템플릿 갱신만으로는 안 바뀝니다. 해당 프로젝트마다 수동 갱신 필요. 자주 쓰는 모델은 가전 규격 마스터에 미리 등록해두시는 게 효율적입니다.

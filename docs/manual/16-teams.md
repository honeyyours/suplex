# 챕터 15. 팀 관리

> 이 챕터를 읽고 나면 — 팀원과 협력업체를 등록하고, 역할별 권한을 부여하며, 신입 사원 온보딩을 빠르게 처리할 수 있게 됩니다.

---

## 15.1 팀원과 협력업체 — 차이

| 종류 | 정의 | 권한 |
|---|---|---|
| **팀원** | Suplex 로그인 가능한 회사 직원 | 역할별 (OWNER/DESIGNER/FIELD) |
| **협력업체** | 발주·일정 대상 거래처 | Suplex 접근 X (마스터 데이터만) |

협력업체는 Suplex에 로그인하지 않습니다. 회사 직원이 그들을 관리하기 위한 마스터 정보일 뿐.

---

## 15.2 팀원 추가

좌측 메뉴 → **팀관리** → **+ 팀원 추가**.

### 입력

| 필드 | 입력 |
|---|---|
| 이름 | 직원 이름 |
| 이메일 | 직원 본인 이메일 (로그인 ID) |
| 역할 | OWNER / DESIGNER / FIELD |
| 부서 (선택) | 디자인부, 시공부 등 |
| 연락처 | 010-XXXX-XXXX |

### 추가 후

임시 비밀번호가 발급됩니다. 직원에게 이메일/카카오톡으로 전달:

> "ABC인테리어 Suplex 계정이 만들어졌습니다.
> ID: 본인 이메일
> 임시 비밀번호: AB12cd34
> 첫 로그인 후 비밀번호를 변경해주세요."

자세한 신입 온보딩 흐름은 [Playbook #6](playbooks/06-team-onboarding.md) 참고.

---

## 15.3 역할별 권한

| 역할 | 권한 |
|---|---|
| **OWNER** | 모든 기능. 팀원 추가/삭제, 회사 정보 수정, 결제 관리 |
| **DESIGNER** | 프로젝트·견적·마감재·발주 작성·수정. 회사 설정 변경 X |
| **FIELD** | 현장보고 입력, 체크리스트 점검, 마감재·일정 조회만. 견적·발주 X |

> 💡 **OWNER는 회사당 최소 1명 필수**. OWNER 본인이 그만두실 때는 다른 직원에게 OWNER 역할을 이양한 후 본인을 삭제하셔야 합니다.

---

## 15.4 비밀번호 초기화

팀원이 비밀번호를 잊었을 때, OWNER가:

**팀관리 → 해당 팀원 행 → "비밀번호 초기화"**

새 임시 비번이 발급됩니다.

> ⚠️ OWNER 본인이 비밀번호를 잊으셨다면 자체 처리 불가. Suplex 운영팀에 문의 부탁드립니다.

---

## 15.5 협력업체 (거래처) 등록

**팀관리 → 협력업체** 또는 **설정 → 협력업체**.

### 입력

| 필드 | 예시 |
|---|---|
| 거래처명 | `XYZ타일` |
| 종류 | 자재 / 인력 / 시공 / 종합 |
| 대표자 | `김대표` |
| 연락처 | `010-XXXX-XXXX` |
| 사업자번호 | `123-45-67890` |
| 주거래 항목 | `타일·세라믹` |
| 결제 조건 | "선금 30% / 잔금 70%" |
| 비고 | 신뢰도, 특이사항 |

이 정보가 등록되어 있으면 발주·지출 입력이 빨라집니다 (자동 채움).

---

## 15.6 초대 토큰

팀원 추가 후 **초대 토큰**을 발행해서 공유 링크로 가입을 유도할 수도 있습니다 (이메일+비번 방식과 별도 옵션).

> ⚠️ 초대 토큰은 **회사 직원**용입니다. 클라이언트(공사 의뢰인)에게는 절대 발행하지 않습니다 — 클라이언트 비접근 원칙.

## 2. 자재별 핵심 발견 + 수정안

### 2.1 타일 — ⭐⭐⭐⭐⭐ (거의 그대로 채택)

**검증된 것**
- 산식 골격(시공㎡ ÷ 1매면적 → ceil → 로스 → 박스)은 한국 실무·표준품셈 정설
- 로스 프리셋 7/10/15/20%는 적합 (오늘의집·자재상 안내와 일치)
- 부위별 가이드(욕실·주방 15%↑)도 실무 정설

**보정 필요**
- **박스당 매수 디폴트 4는 600각 한정**. 규격별 자동 매핑 필요:
  - 300×300 → 16매 / 300×600 → 8매 / 600×600 → 4매 / 600×1200 → 2매
- "로스율 10% 입력 → 박스 올림 후 실질 13~15%" 차이를 UI에 표시 권장

### 2.2 도배 — ⭐⭐⭐⭐⭐ (B 그대로 + 미세 보정)

**검증된 것 (B 산식이 한국 정설로 확정)**
- **LX Z:IN 공식**: "국산 벽지는 1롤당 16~17㎡(약 5평)" → B의 롤당 평수와 일치
- **실평 × 3.5~4배(방), 3배(거실)** = 한국 도배사 통설 → B의 1.65(벽) + 0.65(천장) = 2.3배(실평수) 부합
- **"32평 천장포함 = 실크 15롤" 검증식**: 32×1.65 + 32×0.65 = 73.6평 ÷ 5 = **정확히 15롤** ✅
- 실크 1.06m × 15.6m 규격: LX·현대L&C 공식 표준 일치

**보정 필요**
- **풀 사용량 격차는 B가 과장**: B(실크 1/15kg, 합지 1/8kg = 1.9배 차이) → 실제 통설은 두 종류 다 **0.5~1kg/롤로 비슷**. A의 "1롤=1kg" 가정이 오히려 통념에 가까움
- 광폭합지·소폭 합지 규격이 시리즈별로 다양 → **벽지 규격 마스터 DB**로 두기
- 견적 보수성 위해 **+1~2롤 여유** 디폴트 가능

### 2.5 조명 — ⭐⭐⭐⭐ (W/평 방식) / ⭐⭐⭐ (절대값)

**검증된 것**
- KS A 3011 권장 조도는 **단일값 아닌 범위** (거실 100~200lux 등). A의 lux값은 KS 범위 안 (상한 편향, 보수적)
- LED 효율 90 lm/W: 한국 유통 평균 100~120인데 견적용 안전계수로는 90 적절
- W/평 단위: 한국 시공자 표준 (O0W 거실등 형태로 판매·견적)
- 다운라이트 7W/10W 표준 와트 ✅ (3인치 매입등 7~8W가 표준)

**보정 필요 (B의 절대값이 시공 방식에 종속)**

| 시공 방식 | 거실 W/평 | 비고 |
| --- | --- | --- |
| **방등 1개** | 3~5 | B의 6은 약간 과다 |
| **다운라이트 분산** | 7~18 | B의 6은 **명백히 부족** (10~12 권장) |
| 혼합(방등+포인트) | 8 | 절충안 |

**Suplex 권장: B 기반 + 시공 방식 선택지 추가 + A 학술값 보조 표시**
- 메인 입력: 시공 방식 토글(방등/다운라이트/혼합) → 그에 맞는 W/평 테이블 사용
- 보조 표시: "예상 조도 약 ○○lux (KS 거실 100~200 권장)" 한 줄
- 천장 보정: B식(H/2.4) 채택, H≤2.4면 1.0 고정

### 2.6 에어컨 — ⭐⭐⭐⭐⭐ (4-조합 구조) / ⭐⭐⭐ (계수·테이블)

**검증된 것**
- **시판 평형 enum** [6,7,8,10,...,36]: 삼성·LG 실판매 라인업과 거의 일치 ✅
- **1HP = 2,500 kcal/h** + 1kW = 860 kcal/h: 한국 시공 표준 환산 ✅
- **시스템 4-조합 분기**: 가정 멀티·상업 4Way 카세트 시공 패턴 정확
- 5HP 이상 분산 시공: 시판 단일기 한계(약 14.5kW=5HP) 때문 — 물리 제약

**보정 필요**

| 항목 | A | B | **권장** | 근거 |
| --- | --- | --- | --- | --- |
| 가정 baseKcal | 400 ❌ 높음 | 250 ⚠ 낮음 | **300** | KS 1평=400W=344kcal/h, 한국 평당 300~350 통설 |
| 방향 보정 | 0.95~1.15 ✅ | 0.85~1.25 ❌ 과도 | **0.95~1.10** (A 베이스) | 한국 시공 가이드엔 명시값 거의 없음. B 1.25 과도 |
| 시판 평형 enum | 없음 | 거의 정확 | **B + 9·11 추가** | 벽걸이 9·11평이 표준 라인업에 있음 |
| 평수→방수 | 없음 | 한 단계 낮음 | **≤15:0, ≤22:2, ≤32:3, ≤42:4** | 부동산 표준 25=2/30=3/40=4 |
| 4-조합 분기 | 단순 | 풍부 ✅ | **B 그대로** | 시공 패턴 정확 |

**결론**: B 골격 + 가정 baseKcal 300으로 상향 + 방향 보정 A쪽 완화 + 평형 9·11 추가 + 평수→방수 조정

### 6.2 핵심 발견 3가지

**① 도배 "전용평 × 2.5" 산식이 한국 통설로 확정 ⭐**
- 마미견적서·꾸미고·세민직업학교 **모두 동일 공식**: `도배평 = 전용평 × 2.5`
- B 사이트의 `벽 1.65 + 천장 0.65 = 2.3배`와 거의 일치 (차이 0.2)
- → **B 산식 채택 결정 한 번 더 검증됨**
- → Suplex에 **"간편 모드(×2.5) + 상세 모드(1.65+0.65 분리)"** 토글 추가 (봉기님 결정)

**② 페인트 수성 6 ㎡/L가 한국 4대 제조사·플랫폼 공통값 ⭐**
- LX Z:IN·노루·삼화·오늘의집 모두 **수성 1L = 6㎡** (2회 도장 기준)
- 노루 공식 페이지에 "이론도포량은 손실량 미포함" 명시
- → 이전 §2.4 권고 **7 → 6 ㎡/L로 더 보수적 조정** (봉기님 결정)
- "이론 10 / 실측 6" 두 줄 표기는 노루 공식 입장과 일치

**③ 한국 플랫폼은 두 층으로 갈림 → 간편/상세 모드 토글 설계 근거 ⭐**

| 출처군 | 신뢰도 | 특성 |
| --- | --- | --- |
| 제조사 공식 (LX·노루·삼화·KCC) | ⭐⭐⭐⭐⭐ | TDS·계산기 직접 제공. 절대값 출처로 최우선 |
| 시공자 커뮤니티 (인테리스 info/*) | ⭐⭐⭐⭐⭐ | 시공자 디테일(이론도포량 공식 등). **Suplex B2B 타깃과 매칭** |
| B2C 플랫폼 (마미견적서·오늘의집·꾸미고·집닥) | ⭐⭐⭐⭐ | "5분 견적" 단순 통설(×2.5 등). 디테일 얕음 |
| 일반 계산기 (devcomma·부동산계산기) | ⭐⭐⭐ | 한국 디테일 약함 |

→ **간편 모드(B2C식)** + **상세 모드(B2B식)** 토글이 자연스러운 설계 (봉기님 결정)

### 6.5 신뢰도 인사이트 (영구 지식)

B2C 플랫폼은 **"소비자가 5분 안에 대략적 견적을 보게 하는 게 목적"**이라 ×2.5 같은 단순 통설만 노출. Suplex 같은 **B2B 시공자 타깃**은 인테리스·제조사 TDS 수준의 디테일(부위별 로스·박스당 면적·이론/실측 분리)이 필수.

→ Suplex 자재 산출 모듈의 **간편/상세 토글 구조**가 두 사용자 층(현장 사장님 빠른 견적용 / 견적 담당자 정밀 계산용)을 모두 커버.

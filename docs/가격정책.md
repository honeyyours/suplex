## 4. 베타 정책

- **클로즈 베타** — 초대받은 회사만 (이미 구현된 초대 시스템)
- **2개월 무료** — 결제 정보 등록 X, 카드 등록 X
- 베타 = "프로 등급" 풀 기능 무료 체험 (단, OCR·알림톡은 정식 출시 시 활성)
- **베타 참여자 평생 혜택**: 정식 출시 후 어느 등급으로 가든 **20% 영구 할인**
  - 가격 인상 시에도 동일 비율로 인상하되 **20% 할인은 평생 유지** (약관 명시)

### 베타 후 자동 결제 X
베타 종료일에 자동 청구 시작 X. 사용자가 직접 등급 선택 + 결제 등록 액션 필요.

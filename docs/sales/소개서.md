# 수플렉스 (Suplex)

**인테리어 회사를 위한 통합 운영 도구**
**개인의 능력이 아니라 회사의 능력이 올라가는 도구**
**현장과 사무실, 대표와 직원을 한 줄로 잇습니다.**

---

## 1. 인테리어 회사가 매일 잃는 시간

| 통증 | 결과 |
|---|---|
| 견적·일정·발주가 카톡·엑셀·노션에 흩어져 있다 | 변경 1건당 3~5곳을 수동으로 동기화. 한 곳 빠뜨리면 어긋남 |
| 모바일 달력은 글씨가 안 보여 매번 팝업을 띄워야 한다 | 즉시 판단해야 할 때 시간 손실. 현장에서 자꾸 사무실로 전화해 묻게 됨 |
| 대표·디자이너·현장팀이 같은 정보를 다르게 알고 있다 | 마감재 변경·일정 조정이 일부에게만 전달되어 헷갈림 |
| 자재 발주를 깜빡한다. 시공일이 다가와서야 자재가 없는 걸 발견 | 공기 지연 → 협력업체에 급하게 연락 → 단가 상승 |
| 마감재 결정이 자꾸 바뀌는데 어디까지 확정인지 추적이 안 된다 | 발주서가 옛날 결정 기준이라 자재가 잘못 들어옴 |
| **직원 머리 속에만 있는 노하우** — 현장 환경·자재 노하우·사장님이 챙기던 사전 액션 | **직원이 그만두는 순간 회사 자산이 함께 사라짐** |

수플렉스는 이 통증들을 각 단계에서 정확히 잡고, **일을 쌓을수록 회사 능력이 두꺼워지는** 도구로 작동합니다.

---

## 2. 인테리어 진행 프로세스 × 수플렉스

| 단계 | 디자이너의 작업 | 수플렉스가 받쳐주는 것 |
|---|---|---|
| **1. 상담** | 고객 미팅, 요구사항·예산·일정 수집 | 견적상담 탭에서 공정별로 메모(타일·도배·가구 등). 응대 고객·요청 경로도 함께 기록 |
| **2. 견적** | 비용 산출, 견적서 작성·발송 | 간편 견적(공정 = 그룹) PDF, 회사 기본비율 12개 자동 적용. 가전 모델 검색 → 사이즈 자동 입력 |
| **3. 설계·디자인** | 공간별 마감재·자재 결정 | 마감재 탭(공정별 그룹). 자주 쓰는 자재는 즐겨찾기로 상단 고정. 다른 프로젝트 자재 모달 한 번에 끌어오기 |
| **4. 계획** (공정·자재 사전 준비) | 일정 입력, 협력업체 섭외, 발주 시점 계산 | 일정 입력 시 사전 체크리스트 자동 생성, 발주 데드라인 자동 계산 |
| **5. 시공** | 발주, 현장 진행, 일정 변경 | 마감재 확정 → 발주 탭으로 자연스럽게 이어짐. 발주서·작업자 안내 텍스트 자동 정리 |
| **6. 완공·정산** | 마감 점검, A/S, 데이터 정리 | 프로젝트 기록 누적(피드백·A/S·자재 노하우), JSON 백업 |

> 한국 인테리어 흐름의 4단계는 보통 "계약"이지만, 수플렉스는 클라이언트와 회사 간 계약 자체에는 관여하지 않습니다. 우리가 다루는 건 그 다음의 **계획(공정·자재 사전 준비)** 단계입니다.

## 3. 기능 묶음

### 3-1. 상담·견적

상담 도중 고객 요청은 공정별로 흩어져 나옵니다 — "거실 마루는 강마루로", "도배는 모란디 톤", "주방 싱크대는 한솔 PET". **이걸 종이 노트에 적어두면 견적 작성할 때 다시 펼쳐 헤매야 합니다.**

견적상담 탭은 좌측에 표준 공정(목공·타일·도배 등)을 체크하면 우측에 그 공정 전용 메모란이 펼쳐지는 구조입니다. 공정에 묶이지 않는 정보(가족 구성·예산·주의사항)는 상단 "기본 메모"에. 견적을 작성할 때 같은 화면에서 메모를 보면서 라인을 입력합니다.

가전은 모델명만 검색하면 사이즈가 자동으로 채워집니다(현재 366개+ 모델). 회사가 새로 입력한 모델은 자동으로 누적되어, 다음 프로젝트에선 1초 검색.

### 3-2. 계획·일정

가입 직후부터 회사는 **빈 도구가 아니라 한국 인테리어 표준이 깔린 도구**를 받습니다. 운영팀이 미리 정해둔 5묶음(공정 표시 라벨·자동 인식 키워드·발주 데드라인·사전 체크리스트 어드바이스·견적 가이드)이 신규 회사에 자동 복사되어 처음부터 작동합니다.

일정에 "도배"를 입력하면 도배 D-3에 "벽지 도착 확인", D-1에 "초벌 풀칠 준비"가 체크리스트에 자동 추가됩니다. 발주 화면에는 "타일 D-3 임박" 칩이 색깔로 강조됩니다. **사장님이 챙기던 사전 액션을 신입 직원도 똑같이 따라할 수 있습니다.**

표준이 회사 운영과 안 맞으면 항목별로 활성/비활성 토글. 자기 회사만의 시공 표준을 자체 추가·편집하는 건 **엔터프라이즈 등급에서** 열립니다 (한옥·사찰·카페 전문, 5년+ 노하우 누적된 회사용).

### 3-3. 시공 운영 — 시스템이 텍스트를 자동 정리

시공이 시작되면 **협력업체·작업자에게 안내 메시지를 매번 타이핑하는 일**이 생깁니다. 같은 정보(현장 주소·출입번호·일정·자재 목록)를 카톡방마다 다시 정리해서 보내는 잡일입니다.

수플렉스는 이걸 시스템이 자동으로 정리합니다. 발주 탭에서 항목 체크 → "선택 복사"를 누르면 카톡 친화 형식으로 클립보드에 복사됩니다.

```
[ABC인테리어 / 강남 래미안 304-1502]

[현장 정보]
- 현장명     : 강남 래미안 304-1502
- 주소       : 서울 강남구 도곡동 123, 304동 1502호
- 담당자     : 김미영 (010-XXXX-XXXX)
- 도착 희망일: 2026-05-27
- 특이사항   : 1502호, 엘리베이터 좌측

[발주 항목 — 욕실 타일 시공 자재]
- 타일 600각 화이트 무광       : 15 박스
- 타일 본드 아덱스 X18         : 6 포
- 줄눈재 화이트 5kg            : 4 봉
```

일정 안내도 같은 방식. 일정 탭에서 키워드(예: "전기")로 추출하면 그 공정 일정 + 현장 정보가 자동 정리되어 클립보드로 복사됩니다.

```
[ABC인테리어 / 강남 래미안 304-1502 — 전기 공사 안내]

[일정]
2026-05-18  09:00  설비·전기

[현장 정보]
- 주소     : 서울 강남구 도곡동 123, 304동 1502호
- 출입번호 : 1234*
- 특이사항 : 1502호, 엘리베이터 좌측 / 09시 입주민 양해 완료
```

**프로그램이 잘 짜여 있어서 자동으로 모이는 구조입니다.** 작업자에게 매번 손으로 정리하던 잡일이 한 번의 클릭으로 끝납니다. 모바일에서도 글씨가 직접 보이도록 최적화되어 있어, 현장에서 **사무실에 전화해서 묻지 않고** 바로 확인합니다.

### 3-4. 여러 현장 동시 운영

회사가 잘 굴러가면 동시에 4~10개 현장이 진행됩니다. 엑셀·노션으로 굴리던 회사라면 이 시점부터 정보가 어긋나기 시작합니다. 어느 현장의 일정이 어디인지, 어느 공정에 임박한 발주가 있는지, 어느 현장 디자인 메모가 어디에 있는지 머리 한 곳에 안 들어옵니다.

- **회사 전체 캘린더** — 모든 현장 일정이 색깔별로 한 화면
- **공정별 일정 복사** — "이번 주 전기 작업이 잡힌 모든 현장"을 한 번에 복사 → 전기팀 사장님께 한 번의 카톡으로 4개 현장 안내 끝
- **공정 현황 통합 뷰** — 프로젝트마다 25개 표준 공정 × 견적·마감재·일정·발주 4축 매트릭스. 빠뜨린 항목·미정 항목·임박 데드라인이 색깔로 강조
- **회사 표준 자동 적용** — 4개 현장 모두 같은 D-N 룰·사전 체크리스트로 굴러감. 신입이 굴리든 사장님이 굴리든 같은 품질
- **프로젝트 기록 검색** — 강남 OO아파트 2년 전 견적·A/S·자재 노하우가 검색 한 번으로 호출

### 3-5. 회사 자산화 (엔터프라이즈)

인테리어 노하우는 대부분 사람의 머리 속에 있어, 직원이 그만두는 순간 사라집니다. 수플렉스는 이걸 회사 자산으로 만드는 구조입니다.

- **프로젝트 기록** — 피드백·A/S·자재 노하우·현장 환경 메모 태그별 누적, 검색 한 번으로 다시 꺼냄
- **가전 DB 누적** — 회사가 입력한 모델이 자동 누적, 다음 프로젝트에서 1초 검색
- **회사 표준 시스템 프리셋 자유 편집** (엔터프라이즈) — 어드바이스·D-N 룰·견적 가이드를 회사가 자체 추가·편집. **자기 회사만의 시공 표준을 도구에 새겨넣는 단계**

## 4. 핵심 차별점

| 차별점 | 의미 |
|---|---|
| 개인 능력 → 회사 능력 | 직원 머리 속 노하우가 회사 자산으로. 직원이 떠나도 회사는 남음 |
| 시스템 프리셋 | 가입 첫날부터 표준이 깔린 도구. 빈 도구 X |
| 회사 자산 누적 활용 | 일을 쌓을수록 도구가 더 강해짐 (가전 DB·기록 검색·자재 끌어오기) |
| 여러 현장 동시 운영 | 4~10개 현장 굴려도 회사 전체 캘린더·공정 통합 뷰가 받쳐줌 |
| 시스템이 텍스트를 자동 정리 | 발주서·작업자 안내가 자동 모이는 구조. 매번 타이핑 X |
| 모바일 우선 | 현장에서 사무실에 전화해서 묻지 않고 직접 확인 |
| 클라이언트 비접근 | 회사 영업 정보가 클라이언트에 노출 X. PDF만 발송 |
| 락인 없음 | JSON으로 회사 데이터 통째 백업 가능 |

수플렉스는 한국 인테리어 회사의 실제 업무 흐름에 맞춰 설계되었습니다. 카톡·엑셀을 다 버리라는 게 아닙니다. **수플렉스가 잘 잡아주는 부분만** 안에서 굴리고, 클라이언트·협력업체와의 소통은 기존 채널 그대로 쓰시면 됩니다.

**일을 쌓을수록 도구가 더 강해집니다.** 가전 DB는 두꺼워지고, 회사 표준은 정교해지고, 프로젝트 기록은 회사만의 노하우 자산이 됩니다. 사람이 떠나도 회사는 남습니다.

# 수플렉스 (Suplex)

**인테리어 회사를 위한 통합 운영 도구**
**개인의 능력이 아니라 회사의 능력이 올라가는 도구**
**현장과 사무실, 대표와 직원을 한 줄로 잇습니다.**

---

## 1. 인테리어 회사가 매일 잃는 시간

인테리어 현장의 손실 대부분은 **정보 단절**과 **사람 의존**에서 시작됩니다. 도입 검토 단계의 회사들이 가장 많이 호소하는 통증입니다.

| 통증 | 결과 |
|---|---|
| 견적·일정·발주가 카톡·엑셀·노션에 흩어져 있다 | 변경 1건당 3~5곳을 수동으로 동기화. 한 곳 빠뜨리면 어긋남 |
| 현장에서 일정 확인이 어렵다. 모바일 달력은 글씨가 안 보여 매번 팝업을 띄워야 한다 | 즉시 판단해야 할 때 시간 손실 |
| 대표·디자이너·현장팀이 같은 정보를 다르게 알고 있다 | 마감재 변경·일정 조정이 일부에게만 전달되어 헷갈림 |
| 자재 발주를 깜빡한다. 시공일이 다가와서야 자재가 없는 걸 발견한다 | 공기 지연. 협력업체에 급하게 연락 → 단가 상승 |
| 마감재 결정이 자꾸 바뀌는데 어디까지 확정인지 추적이 안 된다 | 발주서가 옛날 결정 기준이라 자재가 잘못 들어옴 |
| **직원 머리 속에만 있는 노하우** — 어느 아파트는 천장 공간이 좁아 매립등이 안 들어가는지, 어느 모델 수전이 한 달 뒤 헐거워지는지 | **직원이 그만두는 순간 회사 자산이 함께 사라짐** |

수플렉스는 이 통증들을 각 단계에서 정확히 잡아냅니다. 그리고 **일을 쌓을수록 회사 자체의 능력이 두꺼워지는** 도구입니다.

---

## 2. 한국 인테리어 6단계에 끼는 수플렉스

인테리어 표준 흐름은 다음 6단계로 진행됩니다. 각 단계마다 수플렉스가 어떤 일을 자동으로 처리해주는지 정리한 표입니다.

| 단계 | 표준 활동 | 수플렉스가 도와주는 것 |
|---|---|---|
| 1. 현장실측 | 평형·구조 측정 | 프로젝트 카드 생성 — 면적·주소·고객 정보를 한 번에 입력 |
| 2. 설계·디자인 | 마감재·자재 결정 | 가전 366개 모델 자동 검색(사이즈 자동 기입), 마감재 6단계 상태 추적 |
| 3. 견적 | 비용 산출 | 간편 견적 + 상세 견적 두 양식, 회사 기본비율 12개 자동 적용, PDF 출력 |
| **4. 계획** (공정·자재 사전 준비) | 일정·발주 계획 수립 | 일정 입력 시 사전 체크리스트 자동 생성, 발주 데드라인 자동 계산 |
| 5. 시공 | 발주·공정 진행 | 마감재 확정 시 발주 탭으로 자연스럽게 이어짐, 임박한 발주만 D-day 알림, 발주서 자동 작성·복사 |
| 6. 완공 | 정산·이력 정리 | 프로젝트 기록 누적(피드백·A/S·자재 노하우), JSON 백업 |

대표는 한 화면에서 회사 전체의 진행 상황을 봅니다. 디자이너는 마감재·견적에 집중합니다. 현장팀은 모바일에서 오늘 할 일을 즉시 확인합니다. **세 역할이 같은 데이터를** 봅니다.

> 한국 인테리어 표준 흐름의 4단계는 보통 "계약"이지만, 수플렉스는 클라이언트와 회사 간 계약 자체에는 관여하지 않습니다. 우리가 다루는 건 그 다음의 **계획(공정·자재 사전 준비)** 단계입니다.

## 3. 실전 시나리오 — 강남 30평 박상철 고객 리모델링

다음은 인테리어 현장에서 가장 자주 만나는 패턴을 시간순으로 따라간 일곱 컷입니다.

**배경**
- 회사 — ABC인테리어 (강남 일대 4명 팀)
- 견적 담당 — 김미영 사원
- 클라이언트 — 박상철 (40대 4인 가족, 6주 후 입주 예정)
- 프로젝트 — 강남 래미안 304-1502, 30평 전면 리모델링, 예산 약 5,000만 원

---

### 컷 1 — 미팅 직후 프로젝트 등록 + 마감재 빠르게 채움

토요일 오후 2시 현장 미팅이 끝납니다. 김미영 사원은 사무실에 돌아오자마자 수플렉스에 새 프로젝트를 만듭니다.

```
프로젝트명 : 강남 래미안 304-1502 리모델링
고객명     : 박상철
주소       : 서울 강남구 도곡동 123, 304동 1502호
면적       : 30평
시작 예정  : 2026-05-15
종료 예정  : 2026-06-26
```

프로젝트 상세 화면에서 마감재 탭으로 들어갑니다. 비슷한 평형의 이전 프로젝트 데이터를 참고해서 거실·안방·작은방·주방·욕실 마감재를 빠르게 채워 넣습니다. **회사가 진행한 프로젝트가 쌓일수록 새 프로젝트는 더 빨라집니다.**

키보드만으로 빠르게 입력 가능합니다. 행을 클릭하면 펼쳐져서 편집 가능. `Tab`으로 다음 필드, 마지막 필드에서 `Tab`을 누르면 다음 행이 자동으로 펼쳐집니다. 입력 후 1초 뒤 자동 저장됩니다.

---

### 컷 2 — 가전 모델 입력, 사이즈가 자동으로 들어옵니다

박상철 고객은 식기세척기·인덕션·후드를 새로 설치하길 원합니다. 가전은 모델별로 크기가 미세하게 다른데, 잘못 알면 빌트인 가구가 안 맞아 재시공해야 합니다.

마감재 행에 식기세척기를 추가하고 모델명 검색창에 입력합니다.

```
식기세척기 모델 검색: "LG DUE6"
→  LG DUE6BGL2E (BESPOKE 빌트인)
   가로 595 × 세로 815 × 깊이 600 mm
   [선택]
```

선택을 누르면 모델 품번·브랜드·**사이즈가 한 번에 자동 입력**됩니다. 사이즈 출처 URL도 함께 저장되어, 협력업체와 확인이 필요할 때 한 번에 띄워볼 수 있습니다.

수플렉스에는 냉장고·김치냉장고·식기세척기 366개 모델 사이즈가 미리 정리되어 있습니다. 그리고 **사용자가 새 모델을 입력해두면 자동으로 누적되어, 회사만의 가전 DB가 점점 두꺼워집니다.** 다음 프로젝트에서는 그 모델이 바로 검색됩니다.

---

### 컷 3 — 공정 일정, 어디서든 한 화면에 (수플렉스의 메인 기능)

박상철 고객이 견적서를 받고 다음 주에 계약을 결정합니다. 김미영 사원은 6주짜리 공정 일정을 입력합니다.

```
2026-05-15  철거
2026-05-18  설비·전기
2026-05-22  목공
2026-05-29  타일
2026-06-08  도배
2026-06-15  바닥재 시공
2026-06-22  마감 정리
2026-06-26  준공 점검
```

여기서부터가 수플렉스가 다른 프로그램과 다른 지점입니다.

**모바일에서도 공정 이름이 글씨로 직접 보입니다.** 일반 달력 앱은 셀이 좁아서 클릭해야 팝업으로만 글씨가 보이는데, 수플렉스는 인테리어 공정에 맞춰 한 화면에 최대한 많은 정보가 보이도록 설계했습니다. 현장에서 모바일을 꺼내자마자 오늘·이번 주의 공정이 한눈에 들어옵니다.

**모든 팀원이 같은 화면을 봅니다(지출 탭 제외). 다만 각 탭이 누구를 가장 염두에 두고 만들어졌는지가 다릅니다.**

- **일정 탭** — 현장팀 우선. 여러 현장 일정·체크리스트를 넘나들며 확인하기 좋게 최적화. 모바일에서도 한 화면에 최대한 많은 공정이 글씨로 보입니다.
- **마감재 탭** — 디자이너 우선. 많은 마감재를 공정별로 빠르게 추가·제거할 수 있는 UI. 추후 마감재 다시 확인하는 데도 최적.
- **대표** — 모든 데이터를 한 화면에서 쉽게. 지출 탭에서 회사 전체 금전 흐름 확인.

**일정 픽스 — 계획과 확정의 구분**
일정에는 두 단계가 있습니다 — **계획 일정**과 **확정(픽스) 일정**. 처음에는 계획으로 입력해두고, 협력업체 섭외가 끝난 일정만 "픽스"로 바꿉니다. 팀원 모두가 "이 공정은 작업자가 잡혔구나, 저 공정은 아직 섭외 중이구나"를 한 화면에서 구분합니다. **엑셀·노션·달력 앱에는 없는, 인테리어 현장에 맞춰 만든 소통 기능입니다.**

현장에서 일정이 바뀌면 즉시 모바일에서 수정합니다. 사무실 디자이너 화면에 곧바로 반영됩니다. **일정 정보가 개인의 머리 속이 아니라 회사 시스템 안에** 있습니다. 카톡으로 "오늘 도배 끝났어요"라고 알리는 대신 수플렉스에서 체크 한 번. 팀원 모두가 동시에 압니다.

---

### 컷 4 — 시스템이 알아서 챙긴다, D-day 알림 + 사전 체크리스트

인테리어 공사에서 가장 큰 사고는 두 가지입니다.

1. **자재 발주를 깜빡** — 시공일이 다가와서야 자재가 없는 걸 발견. 공기 며칠 지연.
2. **노련한 사장님이 챙기던 사전 액션이 신입에게 빠짐** — "폐기물 업체 미리 섭외", "자재 도착 확인 전화" 등.

수플렉스는 두 가지를 모두 시스템으로 잡습니다.

**D-day 알림 — 임박한 발주만 자동으로**

회사가 미리 등록해둔 공정별 발주 데드라인 규칙을 따라 시스템이 자동으로 어드바이스합니다.

- 타일 시공 시작 7일 전까지 타일 자재 도착
- 도배 시공 시작 5일 전까지 벽지 자재 도착
- 바닥재 시공 시작 7일 전까지 자재 도착

```
타일 시공 시작일: 2026-05-29
타일 발주 마감일: 2026-05-22 (시공 7일 전)

오늘이 2026-05-19 → 발주 화면에 "타일 D-3 임박" 칩 자동 표시
```

발주 메뉴에는 임박한 항목이 통계 카드로 강조됩니다. 마감재가 아직 확정되지 않은 항목은 "모델 미정" 경고로 별도 표시되어, 자재가 정해지지 않은 채 데드라인이 지나가는 일이 없습니다. 회사가 카카오 알림톡 연동(Solapi)을 켜두면, 시공 N일 전에 담당자에게 자동으로 알림톡까지 갑니다.

**사전 체크리스트 — 일정만 입력하면 자동 생성**

일정에 "도배"를 입력하는 순간, 도배 D-3에 "벽지 도착 확인", D-1에 "초벌 풀칠 준비" 같은 사전 액션이 체크리스트에 자동으로 추가됩니다. 회사가 한 번 표준 어드바이스를 등록해두면, 모든 프로젝트가 같은 표준으로 굴러갑니다.

**사장님 머리 속에만 있던 "이쯤 되면 이거 챙겨야 한다" 노하우를 신입 직원도 똑같이 따라할 수 있습니다.** 회사가 일찍 표준을 등록해둘수록, 누가 일을 진행하든 같은 품질이 보장됩니다.

---

### 컷 5 — 발주서 자동 작성, 협력업체에 카톡 형식으로 한 번에

마감재 상태를 "확정"으로 바꾸면 발주 탭에서 자연스럽게 이어집니다. 다른 곳에 다시 입력하지 않아도 됩니다.

발주 화면에서 보낼 항목들을 체크하고 "선택 복사"를 누르면 **발주서가 카톡 친화 형식으로 자동 작성되어 클립보드에 복사**됩니다.

```
[ABC인테리어 / 강남 래미안 304-1502]

[현장 정보]
- 현장명     : 강남 래미안 304-1502
- 주소       : 서울 강남구 도곡동 123, 304동 1502호
- 담당자     : 김미영 (010-XXXX-XXXX)
- 도착 희망일: 2026-05-27
- 특이사항   : 1502호, 엘리베이터 좌측

[발주 항목 — 욕실 타일 시공 자재]
- 타일 600각 화이트 무광       : 15 박스
- 타일 본드 아덱스 X18         : 6 포
- 줄눈재 화이트 5kg            : 4 봉
- 유가 도무스 S라인            : 2 개
- 코너비드 알루미늄 2.5m       : 8 개

확인 후 가능 여부 회신 부탁드립니다. 감사합니다.
```

이 텍스트를 협력업체 사장님 카톡방에 그대로 붙여넣습니다. 협력업체와의 소통 채널을 바꿀 필요 없이, 정리된 정보만 전달됩니다.

---

### 컷 6 — 일정 추출, 작업자에게 보낼 안내를 한 번에

시공이 시작되기 전, 김미영 사원이 전기팀 사장님께 안내를 보내야 합니다. **일정·주소·주의사항을 매번 타이핑하는 일은 인테리어 업무에서 가장 반복되는 잡일** 중 하나입니다.

일정 탭에서 **"일정 추출"**을 누르고 검색창에 `전기`를 입력하면, 그 프로젝트의 전기 공정 일정 + 현장 주소 + 특이사항이 카톡 친화 텍스트로 클립보드에 복사됩니다.

```
[ABC인테리어 / 강남 래미안 304-1502 — 전기 공사 안내]

[공사 일정]
2026-05-18  09:00  설비·전기

[현장 정보]
- 현장명   : 강남 래미안 304-1502
- 주소     : 서울 강남구 도곡동 123, 304동 1502호
- 담당자   : 김미영 (010-XXXX-XXXX)
- 특이사항 : 1502호, 엘리베이터 좌측 / 09시 입주민 양해 완료

잘 부탁드립니다.
```

이 텍스트를 전기팀 사장님 카톡방에 그대로 붙여넣으면 끝입니다. 같은 공정으로 여러 현장이 동시 진행 중이라면, **회사 전체 일정 메뉴**에서 "일정 추출 — 전기"를 누르면 진행 중인 모든 현장의 전기 공사 일정이 한 번에 정리됩니다.

엑셀·노션·달력 앱을 쓸 때 매번 손으로 정리하던 작업이 한 번의 클릭으로 끝납니다.

---

### 컷 7 — 프로젝트 기록, 머리에서 시스템으로

박상철 프로젝트 시공 3주 차. 거실 천장에 매립등을 시공하려는데 천장 속 공간이 8cm밖에 안 됩니다. 김미영 사원은 표면 매입형으로 변경하고, **프로젝트 기록** 탭에 한 줄 기록합니다.

```
[태그: 현장환경]
강남 래미안 304-1502 거실 천장 속 공간 약 8cm.
매립등 시공 시 표면 매입형 권장. 일반 다운라이트 불가.
```

준공 후 한 달, 박상철 고객으로부터 안방 욕실 수전이 헐거워졌다는 A/S 요청이 옵니다. A/S 다녀온 김미영 사원이 또 한 줄 기록.

```
[태그: A/S]
박상철 고객. 안방 욕실 수전 슈베르트 SK-1240, 시공 1개월 후 헐거움.
다음 현장에서 동일 모델 사용 시 체결 토크 강하게.
```

석 달 뒤, ABC인테리어가 같은 아파트 다른 호수 견적을 받습니다. 김미영 사원이 검색창에 `강남 래미안 304`를 입력하자 과거 프로젝트 기록이 즉시 뜹니다. 천장 공간 문제와 욕실 수전 이슈가 사전에 인지된 상태로 견적이 나갑니다.

인테리어 업계의 노하우는 대부분 사람의 머리 속에 있습니다. "A 아파트는 급수 배관에 문제가 많고", "B 아파트는 천장 속 공간이 좁아 매립등 못 한다" 같은 정보는 **직원이 그만두는 순간 사라집니다.** 프로젝트 기록 기능은 이 정보를 회사 자산으로 만듭니다 — 피드백·A/S 내역·자재 발주 노하우·현장 환경 특이사항을 태그별로 기록하고, 검색 한 번으로 다시 꺼냅니다.

추후 AI를 도입해 비슷한 현장을 자동 매칭하고, 과거에 쌓인 노하우를 미리 어드바이스할 계획입니다. **수플렉스를 일찍 도입할수록 회사만의 데이터 자산이 두꺼워집니다.**

## 4. 핵심 차별점

| 차별점 | 의미 |
|---|---|
| **개인 능력 → 회사 능력** | 직원 머리 속 노하우가 회사 시스템 자산으로. 직원이 그만두어도 회사 자산은 남습니다. 일을 쌓을수록 회사 능력이 두꺼워집니다. |
| **사전 체크리스트 자동 생성** | 일정만 입력하면 회사 표준 어드바이스가 작동해 사전 액션이 체크리스트에 자동으로. 노련한 사장님 노하우를 신입도 똑같이 따라할 수 있습니다. |
| **클라이언트 비접근 원칙** | 클라이언트는 수플렉스에 들어오지 않습니다. 회사 영업 정보(매입가·마진율·다른 프로젝트 단가)가 클라이언트에게 보이지 않습니다. PDF만 갑니다. |
| **어디서든 접근, 모바일 우선** | 현장에서 즉시 확인·수정·변경. 모바일에서 공정명이 글씨로 직접 보입니다. |
| **일정 픽스 — 계획과 확정 구분** | 협력업체 섭외 완료된 일정만 픽스 표시. 팀원 모두가 "잡힌 일정"과 "아직 잡히지 않은 일정"을 한 화면에서 구분. |
| **일정·발주 텍스트 추출** | 작업자·협력업체에 보낼 안내를 매번 타이핑할 필요 없음. 일정·주소·특이사항이 한 번에 클립보드로. |
| **JSON 통째 백업** | 회사 전체 데이터를 언제든 한 파일로 내보낼 수 있습니다. 락인 없음. |
| **AI 경리 (예정)** | 추후 통장 데이터 자동 불러오기로 프로젝트별 손익 자동 계산 예정. |

## 5. 도입 첫날 체크리스트

수플렉스에 가입하신 직후 30분~1시간 안에 다음을 마치시면 첫 견적이 빠르게 나갑니다.

- [ ] 회사 정보 입력 (회사명·사업자번호·주소·연락처·이메일)
- [ ] 회사 로고 URL 등록 — 견적서 PDF 헤더에 들어감
- [ ] 견적 기본비율 12개 점검 — 디자인비 10%, 부가세 등을 회사 표준에 맞게
- [ ] 가전 규격 마스터 시드 — 366개 모델 일괄 등록
- [ ] 공정별 발주 데드라인 규칙 등록 — 회사 표준 D-day 룰
- [ ] 사전 체크리스트(어드바이스) 표준 등록 — 회사 표준 노하우 자산화의 시작
- [ ] 팀원 초대 — 대표 / 디자이너 / 현장팀 역할 부여
- [ ] 협력업체 5~10개 등록 — 발주·소통 시 자동완성
- [ ] 기능 표시 설정 점검 — 직원 공유 계정에서 영업 정보 차단 가능

## 6. 다음 행동

도입 검토를 시작하시려면 다음 중 편하신 방법으로 연락 주십시오.

- 데모 요청: __________
- 시범 도입 문의: __________
- 일반 문의: __________

수플렉스는 한국 인테리어 회사의 실제 업무 흐름에 맞춰 설계되었습니다. 회사가 평소에 쓰는 카톡·엑셀을 모두 버리라는 게 아닙니다. **수플렉스가 잘 잡아주는 부분만** 안에서 굴리고, 클라이언트·협력업체와의 소통은 기존 채널 그대로 쓰시면 됩니다.

그리고 **일을 쌓을수록 도구가 더 강해집니다.** 가전 DB는 두꺼워지고, 회사 표준 체크리스트는 정교해지고, 프로젝트 기록은 회사만의 노하우 자산이 됩니다. **사람이 떠나도 회사는 남습니다.**

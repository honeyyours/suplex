## 페르소나 (P1)

> **직원 3~10명 인테리어 회사 사장.** 4~10개 현장 동시 운영. 95% 엑셀+카톡+종이노트로 운영.
> 시공·디자인 전문성 ↑ / 도구·시스템 전문성 ↓. 매출은 늘었으나 사장 본인 시간이 병목.

## JTBD (Job To Be Done) — 한 문장

> 여러 현장을 동시에 굴리면서, 회사가 쌓은 노하우와 정보가 휘발되지 않고, 현장 어디서든 즉시 확인되고, 시스템이 데드라인을 챙겨주게 만들고 싶다 — 그래서 사장 본인이 일일이 챙기지 않아도 회사가 굴러가게.

## 🟦 Customer Profile (사장님 세계)

### Pains (사장님 직접 진술 + 추가)

| # | Pain | 사장님 표현 | 강도 |
|---|---|---|---|
| 1 | **프로젝트 피드백·노하우 휘발** | "남기지도 않았고, 남겼어도 어디 있는지 까먹음. 반영이 안 됨" | ⭐⭐⭐ |
| 2 | **외부 = 사무실 전화 의존** | "현장에서 사무실에 전화해서 '마감재 뭐였지?' '타일 언제?' '마감재 확정됐어?'" | ⭐⭐⭐ |
| 3 | **일정·메모 위치 망각** | "어디 남겼는지 까먹어서 일을 망쳤다" | ⭐⭐ |
| 4 | **발주 데드라인 망실** | "오늘까지 발주했어야 했는데 놓쳤다" | ⭐⭐⭐ |
| 5 | **작업자·자재상 섭외 반복 타이핑** | "현장 주소·상황을 작업자마다 십수번 반복. 자재상 발주까지 하면 더 많음" | ⭐⭐⭐ |
| 6 | 직원 머리속 노하우 증발 | "직원이 그만두면 함께 사라짐" | ⭐⭐ |
| 7 | 변경 1건당 3~5곳 수동 동기화 | 카톡·엑셀·종이 삼중 정리 | ⭐ |
| 8 | 모바일 글씨 안 보임 | 매번 팝업 필요 | ⭐ |

### Gains (얻고 싶은 것)

- **회사 전 직원의 능력이 같이 올라감** (한 명이 나가도 노하우는 회사에 남아 다른 사람이 이어감)
- 같은 인력으로 4~10개 현장을 더 안정적으로 굴림
- 현장에서 즉시 확인 (사무실 의존도 0)
- 시스템이 까먹지 않고 챙겨줌
- 일을 쌓을수록 도구가 더 강해짐 (역방향 락인)
- **효율로 더 많은 매출을 만듦**

### Functional Jobs

상담 → 견적 → 마감재 결정 → 일정 잡기 → 협력업체·자재상 섭외 → 발주 → 시공 운영 → 정산 → A/S → 다음 프로젝트 활용

## 🟥 Value Map (수플렉스가 제공)

### Pain Relievers (Pain ↔ 해소 기능 매핑)

| Pain | 해소 기능 |
|---|---|
| ① 피드백 휘발 | **프로젝트 기록** (태그·검색) + **공정별 마감재 불러오기** + **가전 DB 자동 누적** |
| ② 사무실 전화 의존 | **모바일 우선 설계** + **공정 현황 통합 뷰** (마감재·일정·발주 한 화면) |
| ③ 메모 위치 망각 | **공정 현황 통합 뷰** + **프로젝트 기록 검색** + **견적상담 탭** (공정별 자동 정렬) |
| ④ 발주 데드라인 망실 | **D-day 자동 체크리스트** + **발주 데드라인 자동 계산** |
| ⑤ 섭외 반복 타이핑 | **일정 복사** (키워드→카톡) + **발주서 자동 작성·복사** + **인건비 정산 카톡 복사** |
| ⑥ 직원 그만둠 | **시스템 프리셋** + **회사 표준 D-N 룰** + **프로젝트 기록 회사 자산화** |
| ⑦ 변경 동기화 | 마감재→발주→일정 자동 연결 |
| ⑧ 모바일 가독성 | 모바일 글씨 직접 보임 + 빠른 공종 칩 |

### Gain Creators

- **회사 자산화** ← 프로젝트 기록 검색 + 가전 DB 누적 + 시스템 프리셋
- **회사 전 직원 능력 상승** ← 시스템 프리셋(D-N 룰·키워드·어드바이스 자동) + 같은 화면·같은 데이터
- **다현장 동시 운영** ← 공정 현황 통합 뷰 + 일정 복사 (전기 4개 현장 한 번에)
- **현장 즉시 확인** ← 모바일 우선 + 공정 라벨·키워드 자동 인식
- **시스템이 챙김** ← D-day 알림 + 사전 체크리스트 자동 생성
- **도구가 강해짐** ← 일을 쌓을수록 검색 자산·DB 두꺼워짐

## 🟨 Risk Reversal (도입 망설임 해소)

| 망설임 | 우리 답변 |
|---|---|
| "엑셀이 익숙해서" | **엑셀·카톡 다 버리지 마세요. 수플렉스는 그 사이에 흩어진 정보만 모아줍니다.** |
| "변화가 귀찮아서" | **가입 첫날 30분 셋업.** 빈 도구 X — 한국 인테리어 표준 5묶음(공정·키워드·D-N·어드바이스·견적가이드)이 첫날부터 깔려 있습니다. |
| "데이터 옮기기 귀찮아서" | **옛날 프로젝트 다 옮길 필요 X. 다음 신규 프로젝트부터.** 자재·견적 데이터는 한 번 입력될 때마다 회사 DB에 누적됩니다. JSON 백업으로 락인 없음. |
| (잠재) 월 9.9만원 부담 | 베타 회원 평생 20% → ₩79,200/월. **직원 수 무제한.** |
| (잠재) 직원이 안 따라옴 | **사용법이 쉽고**, 시스템 프리셋이 답을 미리 깔아놨습니다. 신입이 첫 현장 굴리면서 동시에 익힙니다. |

## 🎯 헤드라인 구조 (모든 자료에서 일관 적용)

### 메인 헤드라인 — 표지 / 홈페이지 / 원페이저 Hero
> **"한 사람의 능력이 아니라, 회사 전체의 능력이 올라갑니다."**

### 서브 헤드라인 — 메인 헤드라인 부연
> **효율로 더 많은 매출을 만드세요.**
> 같은 구성원으로 프로젝트 운영 효율을 극대화하고, 담당자가 바뀌어도 업무의 본질과 노하우는 회사의 자산으로 남습니다.

### Problem 슬라이드 헤드라인 — 본문
> **"현장에서 사무실에 전화하지 마세요. 마감재·일정·발주, 까먹지도 잃지도 않습니다."**
